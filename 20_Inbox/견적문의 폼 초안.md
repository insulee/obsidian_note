# 전광판 시스템 견적 문의 폼

## 폼 제목
**전광판 시스템 견적 문의**

---

## 폼 설명
```
다빛솔루션 제품 견적 문의를 위한 양식입니다.
아래 항목을 최대한 자세히 작성해주시면 정확한 견적을 빠르게 안내해드릴 수 있습니다.
작성하신 내용을 검토 후 1~2 영업일 내 연락드리겠습니다.

※ 잘 모르겠거나 폼으로는 표현하기 어려운 사항이 있으시면 
   다빛솔루션으로 직접 연락 주시면 더 빠르고 정확한 상담이 가능합니다.
   전화: [전화번호] | 이메일: [이메일 주소]
```

---

## 섹션 1: 기본 정보

**1. 소속 (회사명/단체명)** *(필수)*
- 단답형 텍스트

**2. 성함** *(필수)*
- 단답형 텍스트

**3. 연락처** *(필수)*
- 단답형 텍스트
- 도움말: "견적 회신을 받으실 휴대폰 번호 또는 유선 전화번호를 입력해주세요"

**4. 이메일** *(선택)*
- 단답형 텍스트
- 도움말: "견적서를 이메일로 받으시길 원하시면 입력해주세요"

---

## 섹션 2: 견적 품목 선택

**5. 견적이 필요한 제품을 선택해주세요 (중복선택 가능)** *(필수)*
- 체크박스 (복수선택)
  - [ ] 전광판 컨트롤러
  - [ ] LED 모듈
  - [ ] 기타 (전광판 키트, 분배보드, 접점보드, 센서 등)

---

## 섹션 3: 전광판 컨트롤러 견적
*(섹션 2에서 "전광판 컨트롤러" 선택 시에만 표시)*

**안내 문구:**
```
전광판 컨트롤러는 연결할 모듈의 단열 수와 필요한 통신방식식에 따라 적합한 모델이 달라집니다.

[컨트롤러별 특징]
• DB500: 최대 40개 모듈 (2단20열 또는 1단40열), LAN 포트 포함, HUB75 2포트
• DB600: 최대 112개 모듈 (4단20열, 최대 8단14열), LAN 포트 포함, HUB75 4포트
• DB502: 최대 40개 모듈 (2단20열 또는 1단40열), LAN 포트 없음, HUB75 2포트

[단열 구성 이해하기]
※ 픽셀: 전광판에 있는 LED 밝기 소자 1개
※ 단열: 단은 세로 방향, 열은 가로 방향을 의미합니다

예시)
- 가로 32픽셀 x 세로 16픽셀 전광판 = 1단 2열
- 가로 64픽셀 x 세로 32픽셀 전광판 = 2단 4열
- 가로 320픽셀 x 세로 64픽셀 전광판 = 4단 20열

※ 모듈 구성은 모듈 크기에 따라 달라지므로, 픽셀 수나 단열 수로 작성해주시면 됩니다.

[필요한 정보]
1. 연결하려는 전광판의 단열 구성 (예: 4단 10열, 2단 20열 등)
   또는 전체 픽셀 수 (예: 가로 320픽셀 x 세로 64픽셀)
2. 필요한 통신 방식 (TTL, RS232, Ethernet, RS485, WiFi/BLE 등)
3. 선호하는 컨트롤러 모델 (있는 경우)
4. 필요 수량
```

**6. 전광판 컨트롤러 견적 상세 내용** *(필수)*
- 장문형 텍스트
- 안내 문구: "위 안내를 참고하여 연결하려는 전광판의 정보와 필요한 사양을 자세히 작성해주세요. 잘 모르겠거나 추가 문의가 필요하시면 다빛솔루션으로 직접 연락 주시기 바랍니다."
