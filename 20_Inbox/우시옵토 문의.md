### 500S 페이지 메시지 표출 문제

#### 케이스 1: v2.5.3 → 다른 버전
- 2.5.3에서 페이지 메시지 1개 전송 ("2.5.3페이지1")
- 2.4.1로 변경 시: **정상** ("2.5.3페이지1" 그대로 표출)
- 3.3.5로 변경 시: **비정상** (이상한 문자 표출)

#### 케이스 2: v3.3.5 → 다른 버전
- 3.3.5에서 페이지 메시지 1개 전송 ("3.3.5페이지1")
- 2.4.1로 변경 시: **비정상** (아무것도 표출 안됨)
- 2.4.1에서 새로 메시지 전송 ("2.4.1페이지1")
- 2.5.3으로 변경 시: **정상** ("2.4.1페이지1" 그대로 표출)
- 3.3.5로 변경 시: **비정상** ("3.3.5페이지1"이 복원되어 표출됨)
- 3.1.4도 같은 문제

#### 케이스 3: 페이지 메시지 개수 설정
- 3.3.5에서 페이지 메시지 3개 등록 및 전송
- 2.4.1로 변경 시: 2.4.1에서 전송했던 1개 메시지만 표출, 나머지 2개는 빈칸으로 순환 표출

#### 기타 사항
- 페이지 메시지 개수 등록 명령은 펌웨어 변경 시에도 정상 적용
- 메시지 초기화 명령은 펌웨어 변경 시에도 정상 적용

### 아이파킹 시스템에서 페이지메시지 표출 오류
- 주차관제 시스템에서 v2.4.1에서는 되는데 v3.3.4에서는 안되는 문제(다빛솔루션 프로그램으로는 모두 표출 가능)
- 정상표출: 외부에서 들어오는 실시간 메세지(인신된 차량번호나 주차금액)는 2줄 표출됨
- 비정상: 아이파킹 시스템을 통한 고정메세지 보낼시 윗줄만 표기(예:00아파트,어서오세요 등의 상시로 표출되는 문구류)
