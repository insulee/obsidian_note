## 배경 및 목적

### 기존 방식의 문제점

- DB600에서 16 DUTY 모듈(빛샘모듈) 연결 시 **3개 보드 연결 필요**
    - DB600 → BUF21-HUB75-OUT34P V3 → BUF51-3C34-3C206Y → **DIBD-SEC1600-16D-VS** → 빛샘모듈

### 개선 방안

- **BUF28-HUB75-3C16p-VS16D** 신규 제작으로 **2단계로 단순화**
    - DB600/DB500 → **BUF28-HUB75-3C16p-VS16D** → 빛샘모듈
    - 16핀 HUB75 케이블에서 직접 변환보드 연결

## 테스트 결과

### 1. 표출신호 설정 (중요)

| 변환보드                         | 표출신호             |
| ---------------------------- | ---------------- |
| DIBD-SEC1600-16D-VS (기존)     | `16D-P16D1S10-1` |
| BUF28-HUB75-3C16p-VS16D (신규) | `16D-P16D1S11` ✓ |

**→ 신규 변환보드 사용 시 표출신호를 `16D-P16D1S11`로 설정**

### 2. 다단 구성 연결 방법

#### DB600 

**4단 구성 예시:**

```
DB600 HUB75-1 → BUF28-HUB75-3C16p-VS16D  ─ OUT1 → 1단
						                └─ OUT2 → 2단

DB600 HUB75-2 → BUF28-HUB75-3C16p-VS16D  ─ OUT1 → 3단
						                └─ OUT2 → 4단
```

- 변환보드의 OUT1/OUT2로 **2단 출력**
- 4단 구성 시 변환보드 2개 사용

#### DB500 

**2단 구성 예시:**
```
DB500 HUB75-1  → BUF28-HUB75-3C16p-VS16D  ─ OUT1 → 1단
						                 └─ OUT2 → 2단
```
