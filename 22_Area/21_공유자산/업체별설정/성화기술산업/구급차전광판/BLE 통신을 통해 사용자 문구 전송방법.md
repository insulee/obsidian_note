## 성화기술산업 - 구급차전광판 프로젝트

- 본 문서는 ASCII 프로토콜을 이용해 컨트롤러에 사용자 문구 11번, 12번을 전송하는 법을 안내
- 11번 문구는 페이지 메시지 1번으로, 12번 문구는 페이지 메시지 2번으로 전송
- 전송 후, 컨트롤박스에서 11번, 12번을 누르면 해당 문구가 표출 
- ASCII 프로토콜 문서 https://buly.kr/FhNljot
#### 사용자 문구 전송 방법
- 11번 전송: `![001/P0000/F0207/T0.......!]`
	- `001`: 페이지 문구 선언
	- `/P0000`: 페이지 1번, 섹션 1번(기본값)
	- `/F0307`: UTF-8코드/유니코드폰트, 폰트크기32x32

- 12번 전송: `![001/P0100/F0307/T1.......!]`
	- `001`: 페이지 문구 선언
	- `/P0100`: 페이지 2번, 섹션 1번(기본값)
	- `/F0307`: UTF-8코드/유니코드폰트, 폰트크기32x32
- 설명
	- 사용자문구 전송
		- 11번 문구: 페이지 1번
		- 12번 문구: 페이지 2번
	- 폰트
		- 폰트는 사전에 컨트롤러에 전송 필수
		- 폰트그룹 1번에 한글조합형폰트, 2번에 유니코드폰트를 전송했다면
			- `/T0`: 폰트그룹 1번
			- `/T1`: 폰트그룹 2번
		- 다빛솔루션이 제공한 폰트외에 폰트를 이용하려면 자체 제작
	- 문자 인코딩
		- 휴대폰 전송시 UTF-8로 전송하므로 `/F02` 또는 `/F03`으로 전송
		- 표출하는 폰트가 한글조합형이냐(`/F02`) 유니코드냐(`/F03`)에 따라 선택
	- 폰트 크기
		- 한글기준 16x16 폰트: `03`으로 전송
		- 한글기준 32x32 폰트: `07`로 전송
		- LED 모듈이 2단8열(32x128픽셀)로 1~10번 문구와 동일한 크기는 32x32
- 이외에 프로토콜 문서를 참고하여 입/퇴장효과, 효과속도, 유지시간, 문자색상 등을 전송

</br> 

#### 참고
- /Pxxyy 
	- xx: 페이지 번호
	- yy: 섹션 번호
- /Fxxyy: 문자인코딩 및 폰트
	- xx: 문자 인코딩 및 폰트
		- 시스템 -- A --> 컨트롤러 -- B --> LED 모듈
		- `00` - A: EUC-KR코드(2Byte), B: 한글 조합형 폰트(2Byte)
		- `01` - A: UTF-16코드(2Byte), B: 유니코드 폰트(2Byte)
		- `02` - A: UTF-8코드(3Byte), B: 한글 조합형 폰트(2Byte)
		- `03` - A: UTF-8코드(3Byte), B: 유니코드 폰트(2Byte)
	- yy: 폰트크기 (가로픽셀x세로픽셀)
		- `03` - 영문자/숫자: 8x16 / 한글: 16x16
		- `07` - 영문자/숫자: 16x32 / 한글: 32x32
