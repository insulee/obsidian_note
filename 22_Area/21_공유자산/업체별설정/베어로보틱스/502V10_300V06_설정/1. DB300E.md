### DB300E(ESP32) Binary File Update
- 업로드 파일
	- Bootloader 펌웨어: DB300-BOOT V1.0.0.bin
	- User 펌웨어: DB300E_V2.0.4.bin
- DB300E를 JP1 단락 후 TTL로 전원공급
- flash_download_tool_3.9.6-DB502_PV10.exe 실행 
	- ChipType: `ESP32`
	- WorkMode: `Develop`
	- LoadMode: `UART`
	- `OK`
		- ![[베어_300_esp32.png]]
	- DB300-BOOT V1.0.0.bin @0x00000  
	- DB300E_V2.0.4.bin @0x10000
	- `START`
		- ![[베어_300_esp32_v10.png]]

### DB300E Mac Address 입력
#### (중요) 맥주소 전용 다빛채 (DabitChe_V2.2.0_BearRobotics_mac) 사용
- 출력한 Mac Address 바코드를 DB300E 뒷면에 부착
- DB300E를 TTL로 연결
- 다빛채 > `메뉴모음` > 메뉴모음 창 > `통신설정` > Serial 통신 > `포트열기` > `컨트롤러 연결`
	- F10 > `MacAddress 설정` > `DB300` `부착한 Mac Address` > 전송 > 읽기 로 전송 확인
		- ![[베어_300_macaddress_v10.png]]

### DB300E 초기값 설정
#### (중요) 300 전용 다빛넷 (DabitNet_S_V1.1.0_300) 사용
- DB300E를 LAN으로 연결
- DABITNet > `Search` > DB300E의 Mac Address 선택
- `Set Default IP`를 눌러 아래의 항목이 적용됐는지 확인
	- Name: DB300E
	- LocalIP: 10.10.127.10
	- Port: 5000
	- SubnetMask: 255.255.255.0
	- Gateway: 10.10.127.1
	- ServerIP=10.10.127.1
	- ServerPort=5000
		- ![[베어_300_dabitnet_v10.png]]
	- `Set`
	- `Search` > 정상적으로 변경됐는지 확인
