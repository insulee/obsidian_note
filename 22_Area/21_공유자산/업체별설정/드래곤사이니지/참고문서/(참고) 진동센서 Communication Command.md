## 1. 인터페이스 사양
- **물리층**: RS-232C  
- **전송 속도**: 4800/9600/19200 bps (기본 9600 bps)  
- **데이터 비트/패리티/스톱 비트**: 8 bit, 패리티 없음(N), 1 bit (8N1)  

## 2. 명령어 포맷
- **PC → 3233**: 1 Byte ASCII 대문자(CMD)  
- **3233 → PC**: ASCII 문자열(가변 길이) + CR(0x0D), LF(0x0A)

## 5. 사전 준비 (원격 모드)
1. **원격 모드 선택**  
   - Menu → Meas Mode → **Remote**  
   - Interval → **Single : off**  
   - OUT Ch → **X**  
   - LCD cont → (조정값 입력)  
   - Date y/m/d → **00/01/01**  
   - Time → **00:00:00**  
2. 설정 완료 시 상단에 `[Remote]` 표시, **Menu** 키 외 모든 버튼 잠김  
3. 전원 재투입 시 자동 원격 모드 진입  
4. 해제 시 Menu → Meas Mode → **Manual**

## 6. 통신 타이밍
1. **전원 투입 후 대기**  
   - 3233 전원 ON → 명령 대기 상태  
   - PC 명령 전송 → 3233 응답/데이터 전송  
2. **구성 파일 전송**  
   - PC → `F0nnnn`~`F4nnnn`  
   - 3233 → `f0`~`f4` (각 설정 완료)  
3. **측정 시작/정지**  
   - PC → `S` → 3233 → `s`  
   - PC → `E` → 3233 → `e`  
4. **데이터 취득 알림**  
   - 완료 시 3233 → `r` 전송  
   - Repeat 모드 시 주기마다 `r` 전송  
5. **Lp값 획득**  
   - PC → `P1` → 3233 → `X110.0\r\n`  
   - PC → `P0` → 3233 → `X…\r\nY…\r\nZ…\r\n`
