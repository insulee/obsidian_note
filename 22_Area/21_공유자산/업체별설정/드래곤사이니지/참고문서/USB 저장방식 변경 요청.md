### 1. 기존 프로세스
- 다빛솔루션에서는 미세먼지, 소음 제품 출고시 아래의 내용 세팅
	- 펌웨어 업데이트
	- 보드기능설정 > 제품별 Baud Rate 설정(센서, GPS)
	- 표출신호
- 드래곤사이니지에서는 폰트, 배경이미지 등이 들어있는 USB를 꽂아서 최종 고객에 납품
### 2. 요청 사항
- 현재 상황
    - 폰트, 이미지, 스케줄, 설정 등이 USB에 저장 → USB 분리 시 전광판 동작 안함
    - 드래곤사이니지 현장의 고객이 USB를 다루는 과정에서 관리가 어려움
- 요청
    1. **USB에는 측정값(TXT)만 저장**
    2. **폰트, 배경이미지, 스케줄, 설정 등은 모두 DB502S 메모리로 저장**
    3. **USB 분리 후에도 전광판이 측정값을 정상 표출**
- **기존의 미세먼지, 소음 전광판 외에 추가 개발할 진동, 초미세먼지도 동일하게 적용 요청**

### 3. 개발 완료시 논의 사항
- 다빛솔루션에서 출고시 폰트, 배경이미지(단위), 전광판 설정등을 해서 출고 (비용추가 안내)
- 또는 기존 방식으로 출고 후 드래곤사이니지에서 다빛채로 폰트, 배경이미지 전송
- 개발 완료시점에서 관련 내용 전달하고 결정
