#### 프로그램 위치
- dfu 프로그램
	- staff > 70. Install > dfusecV1_11 > DFUSec.exe
- LPC USB 드라이버
	- staff > 70. Install > dfusecV1_11 > lpc18xx_43xx_winusb_drivers_v1.00

#### 하드웨어 설정
- 컨트롤러와 PC를 USB 포트를 이용해 연결
- JP1(2핀 점퍼핀)을 단락시키고 전원을 재공급
- 컨트롤러의 PWR LED에만 빨간불이 들어오면 준비 완료

#### dfu 프로그램 설정
- DFUSec.exe 실행
- dfu 프로그램 우측하단에 `시작` 버튼이 비활성화면 **LPC USB 드라이버 설치**
	- 장치관리자 > 기타장치 > LpcDevice를 오른쪽버튼 클릭 > `드라이브 업데이트` 선택
	- lpc18xx_43xx_winusb_drivers_v1.00 폴더 선택 (하위 폴더 포함 옵션 체크 확인)
	- `다음`을 클릭해 설치 진행
- Program mode
	- Step: `0`
		- Algo: `HDR\iram_dfu_util_any.bin.hdr`
		- File
			- DB500: `BOOT4330_DIBD500_V2.0.0.bin`
			- DB600: `BOOT4330_DIBD600_V2.0.0.bin`
		- Address: `0x14000000`
		- Size: `0x00020000`
		- Param: `0x00000000`
		- Device erase Options: `No erase`
		- Operation after program cycle complete: `Do nothing`
		- Verity image after programming by reading it back 체크
		- Compute/Insert chechsum in image before programming 체크
	-  Step: `1`
		- Algo: `HDR\iram_dfu_util_any.bin.hdr`
		- File: 각 컨트롤러별 펌웨어
		- Address: `0x143c0000`
		- Size: `0x00030000`
		- Param: `0x00000000`
		- Device erase Options: `No erase`
		- Operation after program cycle complete: `Reset`
		- Verity image after programming by reading it back 체크
		- Compute/Insert chechsum in image before programming 체크
	- `Start`
