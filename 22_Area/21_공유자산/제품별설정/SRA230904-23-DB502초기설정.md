#### 프로그램 위치
- flash download 프로그램
	- staff > 70. Install > flash_download_tool_3.9.6 > flash_download_tool_3.9.6.exe

#### 하드웨어 설정
- 컨트롤러와 PC를 TTL 포트를 이용해 연결
- JP1(2핀 점퍼핀)을 단락시키고 전원을 재공급
- 컨트롤러의 PWR LED에만 빨간불이 들어오면 준비 완료

#### flash download 프로그램 설정
- flash_download_tool_3.9.6.exe 실행
	- DOWNLOAD TOOL MODE
		- ChipType: `ESP32-S3`
		- WorkMode: `Develop`
		- LoadMode: `UART`
		- `OK`
	- ESP32S3 FLASH DOWNLOAD TOOL V3.9.6
		- `DB502-BOOT V2.0.0.bin` @ `0x000000`
		- `DB502S_08C_02R020C_V2.1.0.bin` @ `0x010000`
		- `SPI SPEED: `80MHz`
		- SPI MODE: `QIO`
		- DoNotChgBin 체크`
		- `START`
	
#### Mac Address 입력
- 출력한 Mac Address 바코드를 컨트롤러 뒷면에 부착
- 컨트롤러와 PC를 TTL로 연결
- 다빛채 > `메뉴모음` > 메뉴모음 창 > `통신설정` > Serial 통신 > `포트열기` > `컨트롤러 연결`
	-  F10 > `MacAddress 설정` 
	- MacAddress 설정 창 > `DABIT 502` > `컨트롤러에 부착한 Mac Address` > `전송`
	- `읽기`를 눌러 전송 확인
	- `닫기`
