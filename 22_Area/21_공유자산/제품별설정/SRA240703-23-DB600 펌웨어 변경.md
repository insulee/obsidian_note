
> [!Abstract] DB600시리즈 펌웨어 업데이트
> - DB600 시리즈는 DB600S, DB600T, DB600U, DB600V, DB600W 총 5가지 모델
> - 각 모델의 펌웨어 변경은 기존의 방식과 동일
> - 다빛채에서 모델 간 펌웨어 변경방식에 관해 안내

#### DabitChe.ini 변경
- DabitChe 폴더 안에 DabitChe.ini 실행
- ShowAutoUpGrade=0 -> `1` 변경 후 저장
- (참고) ShowMacaddress=0 -> 1 (F10: mac address 확인)

#### 펌웨어 변경
- 다빛채 > `메뉴모음 창` > `펌웨어업그레이드`
	- 펌웨어업그레이드 > `읽기` > `열기` 
		- Open Firmware File > `.bin` > `열기`
	- `닫기`
	- `F12`
	- 자동전송목록 > Firmware Upgrade 체크 > `자동전송`
