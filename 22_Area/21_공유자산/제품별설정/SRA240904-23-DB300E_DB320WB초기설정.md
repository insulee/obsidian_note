#### 프로그램 위치
- flash download 프로그램
	- staff > 70. Install > flash_download_tool_3.9.6 > flash_download_tool_3.9.6.exe

#### 하드웨어 설정
- 컨버터와 PC를 TTL 포트를 이용해 연결
- JP1(2핀 점퍼핀)을 단락시키고 전원을 재공급
- 컨버터의 PWR LED에만 빨간불이 들어오면 준비 완료

#### flash download 프로그램 설정
- flash_download_tool_3.9.6.exe 실행
	- DOWNLOAD TOOL MODE
		- ChipType: `ESP32`
		- WorkMode: `Develop`
		- LoadMode: `UART`
		- `OK`
	- ESP32 FLASH DOWNLOAD TOOL V3.9.6
		- 공통: `DB300-BOOT V2.0.0.bin` @`0x00000`
		- DB300E: `DB300E_V2.0.4.bin` @`0x10000`
		- DB320WB: `DB320WB_V2.1.1.bin` @`0x10000`
		- `SPI SPEED: `80MHz`
		- SPI MODE: `QIO`
		- DoNotChgBin 체크`
		- `START`
- 부트로더와 펌웨어 업로드시 기본 IP값이 컨버터에 저장
	- 추가 IP 세팅이 필요하면 아래의 내용을 참고
		- [SET Default IP 설정](https://publish.obsidian.md/dabitdocs/3.+%EC%A0%84%EB%AC%B8%EC%9E%90%EB%A3%8C/SET+Default+IP+%EC%84%A4%EC%A0%95)
		- [[1. DB300E]] > DB300E 초기값 설정

#### Mac Address 입력
- 출력한 Mac Address 바코드를 컨버터 뒷면에 부착
- 컨버터와 PC를 TTL로 연결
- 다빛채 > `메뉴모음` > 메뉴모음 창 > `통신설정` > Serial 통신 > `포트열기` > `컨트롤러 연결`
	- F10 > `MacAddress 설정` 
	- MacAddress 설정 창 > `DABIT 300` > `컨버터에 부착한 Mac Address` > `전송`
	- `읽기`를 눌러 전송 확인
	- `닫기`
