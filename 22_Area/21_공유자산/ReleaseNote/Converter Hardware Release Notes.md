#### DB320(Wifi Converter)
- ##### PCB V3.1 (2024-10-11)
	- DB500/600/502/300/320 보드에 장착된 LED 명칭 및 부품 번호 통일
		- LED1 (PWR): 전원 확인 용 적색 LED
		- LED2 (S1): 통신 및 표출 효과를 담당하는 메인 CPU 동작 확인 용 녹색 LED (500ms 간격 점멸)
		- LED3 (S2): Display용 CPU 동작 확인용 녹색 LED, 메인 CPU에서 구현된 이미지의 실시간 전송 확인 (500ms 간격 점멸)
		- LED4 (RX1): RS232 통신 수신 확인 용 녹색 LED
		- LED5 (TX1): RS232 통신 송신 확인 용 적색 LED
		- LED6 (TX2): TTL 통신 송신 확인 용 적색 LED
		- LED7 (RX2): TTL 통신 수신 확인 용 녹색 LED
- ##### PCB V3 (2024-06-11)
	- **하드웨어 구성**
		- 조립 편의성 개선: 
			- C13 size변경: 1608->1005
			- C5 size 변경: 1005->1608
			- 5267 connector series J1(2PIN),J2, J4, J6(5PIN) pin hole size 변경: 0.9mm->0.95mm
- ##### PCB V2 (2024-05-28)
	- **하드웨어 구성**
		- C13 Top면에 배치
		- 조립 편의성을 위해서 H1, H2, H3 기구 HOLE PAD 변경
- ##### PCB V1 (2024-04-29)
	- **하드웨어 구성**
		- 통신 방식: WIFI, RS232, TTL, BLE(Bluetooth Low Energy)
		- 무선 인증서 취득 제품: 
			- 이 제품은 필요한 무선 인증(WIFI, BLE)을 모두 취득하여, 규정을 준수하는 안전하고 신뢰할 수 있는 사용이 가능합니다.
		- 블루투스 설정 지원: 
			- 이 제품은 블루투스 연결을 통해 무선랜(WIFI) 설정과 연결된 다빛 컨트롤러의 표출 화면을 쉽게 조정할 수 있습니다
	- **적용 가능한 펌웨어**
		- 2024-06-01 이후의 펌웨어 버전 1.0.0 부터 지원
		- 최신 펌웨어: DB320_WIFI_BLE_V1.0.0.bin
