### 컨트롤러 출시 이력(Release history)
#### DB502(Serial 통신용)
- ##### PCB V13 (2024-11-07)
	- V12.1에서 BOM상의 부품 값만 변경한 같은 제품.
- ##### PCB V12.1 (2024-10-14)
	- DB500/600/502 보드에 장착된 LED 명칭 및 부품번호 통일
		- LED1 (PWR): 전원 확인 용 적색 LED
		- LED2 (S1): 통신 및 표출 효과를 담당하는 메인 CPU 동작 확인 용 녹색 LED (500ms 간격 점멸)
		- LED3 (S2): Display용 CPU 동작 확인용 녹색 LED, 메인 CPU에서 구현된 이미지의 실시간 전송 확인 (500ms 간격 점멸)
		- LED4 (RX1): RS232 통신 수신 확인 용 녹색 LED
		- LED5 (TX1): RS232 통신 송신 확인 용 적색 LED
		- LED6 (TX2): TTL 통신 송신 확인 용 적색 LED
		- LED7 (RX2): TTL 통신 수신 확인 용 녹색 LED
- ##### PCB V12 (2024-06-11)
	- **하드웨어 변경사항**
		- 조립 편의성 개선: 
			- U1(ESP32-S3) PAD size 변경 
				- 직사각 PAD는 0.4mm x 0.8mm -> 0.45mm x 0.9mm 
				- 정사각 PAD는 0.8mm->0.9mm
			- 5267 connector series J2, J3, J4 hole size 변경 0.9mm->0.95mm
			- 기구 HOLE PAD 변경
- ##### PCB V11 (2024-04-24)
	- **하드웨어 변경사항**
		- RS232(J2 4핀 커넥터) 신호 라인의 정전기 보호를 위해 ESD protection diode를 추가.
		- TTL/RS485 통신(BH1 10핀 커넥터) 신호 라인의 정전기 보호를 위해 ESD protection diode를 추가.
- ##### PCB V10 (2023-07-25)
	- **하드웨어 변경사항**
		- micro 5핀 커넥터를 USB-C type 커넥터로 변경
			- 이 변경은 유럽 연합을 시작으로 전세계적으로 모바일 기기에 USB Type-C 단자 탑재를 의무화하는 법안이 통과됨에 따라 이루어졌습니다.
		- LED 모듈 제어 신호의 구동 전압을 3.3V에서 5V로 변경
			- 이 변경으로 커넥터와 LED 모듈을 연결하는 케이블의 길이가 10미터 이상일 때도 잡음의 영향을 크게 줄일 수 있습니다.
		- 2핀 커넥터(J1) 및 4핀 커넥터(J2, J3, J4)의 방향을 180도 변경하여 케이스 설계 시 앵글 타입 커넥터의 장착을 용이하게 함
		- 신뢰도가 낮은 LED 모듈에서 컨트롤러의 전원만 리셋할 때 발생하는 미세 잡음 제거
		- 1/32Duty의 엘이디모듈 표출 가능
		- SD 카드 장착으로 SD 메모리에 더 많은 이미지 저장 가능
		- 최적의 신호 라인 배치를 통해 디스플레이 화질 개선
	- **적용 가능한 펌웨어**
		- 2023-11-06 이후의 펌웨어 버전 2.0.0 부터 지원
		- 최신 펌웨어: DIBD502S_08C_02R016C_V2.2.0.bin
- ##### PCB V6 (2022-10-20)
	- **하드웨어 변경사항**
		-  통신 방식: RS232, TTL, RS485(옵션)
		- LED모듈 제어용 커넥터 : HUB75 16pin 2ea
	- **적용 가능한 펌웨어**
		- 2023-03-13 이후의 펌웨어 버전 8.98부터 지원
		- 최신 펌웨어: DB502S_08C_02R016C_V9.16.bin

		- 필수 부품만을 선별해 부품 및 보드 크기 최소화 및 합리적인 가격 제공
