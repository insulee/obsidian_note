# DB600T + BUF28-HUB75-3C16p-VS16D 연결 매뉴얼
---
## 0. 빛샘전자 VS096T110-0(16 DUTY) 4단 8열 구성

## 1. 준비물

- DB600T 컨트롤러
- BUF28-HUB75-3C16p-VS16D 변환보드 **2개**
- 빛샘전자 VS096T110-0 모듈 **32개** (4단 × 8열)
- HUB75 16핀 케이블 + 빛샘전자 필름 케이블



## 2. 하드웨어 연결

```
DB600T HUB75-1 → BUF28 변환보드 #1
                    ├─ OUT1 → 1단 (모듈 8개)
                    └─ OUT2 → 2단 (모듈 8개)

DB600T HUB75-2 → BUF28 변환보드 #2
                    ├─ OUT1 → 3단 (모듈 8개)
                    └─ OUT2 → 4단 (모듈 8개)
```



## 3. DB600T 설정값

| 항목           | 설정값                  |
| ------------ | -------------------- |
| **세로**       | `4`                  |
| **가로**       | `8`                  |
| **모듈배열방법**   | `가로형`                |
| **표출신호**     | `16D-P16D1S11`       |
| BH1 Function | `8pin Input(Number)` |
