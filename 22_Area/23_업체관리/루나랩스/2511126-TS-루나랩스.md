# 실시간 메시지 복귀 오류 - 문의 및 해결 방법

## 📋 고객 문의 내용

### 현재 상황

**평상시 메시지 (2개 섹션으로 구성)**
```
![000/P0000/X0024/Y0004/E0606/S2002/C2제각말 5-4 아파트!]
![000/P0001/X0024/Y0408/C3어서오세요!]
```

**차량 감지 시 실시간 송출**
```
![000/P0000/D0101/S2010/T0/C212가1234  /C1등록차량 !]
```

### 문제 증상

실시간 송출 시간이 끝나면 다시 평상시 메시지로 복귀되어야 하는데
- 위 섹션: 사라져서 빈 문구가 됨
- 아래 섹션: 그대로 "어서오세요"를 유지하고 있음

## 💡 답변 내용

### ○ 현재 문제 상황

고객님께서 보내주신 메시지를 확인해보니,
- 평상시 메시지를 실시간 메시지(![000/!])로 전송하고 계십니다.
- 차량 감지 시 같은 섹션(P0000)에 차량번호가 표시되면서 "제각말 5-4 아파트" 메시지를 덮어씁니다.
- /S2010 유지시간이 끝나면 실시간 메시지가 사라지면서 빈 화면이 됩니다.
- "어서오세요"(P0001 섹션)는 차량 메시지로 덮어쓴 적이 없어서 그대로 계속 표시되는 것입니다.

### ○ 해결 방법 안내

평상시 메시지는 페이지 메시지(![001/!])로 전송하셔야 합니다.

페이지 메시지는 ROM에 저장되어 항상 유지되기 때문에, 실시간 메시지 표시가 끝나면 평상시 메시지로 복귀합니다.

### ○ 설정 방법

**1. 다빛솔루션 프로그램 설정**

"페이지메시지 개수설정"을 1로 설정해주세요.

**2. 평상시 메시지를 페이지 메시지로 전송**
```
![001/P0000/X0024/Y0004/E0606/S2002/C2제각말 5-4 아파트!]
![001/P0001/X0024/Y0408/C3어서오세요!]
```
기존 `![000/` 을 `![001/` 로만 변경하시면 됩니다.

**3. 차량 감지 시 실시간 메시지 전송**
```
![000/P0000/D0101/S2010/T0/C212가1234  /C1등록차량 !]
```

### ○ 아래와 같이 동작합니다.

- 평소: "제각말 5-4 아파트" (왼쪽으로이동) + "어서오세요" (정지) 표시
- 차량 감지: "212가1234 등록차량" 표시
- 유지시간 종료: 자동으로 원래 메시지("제각말 5-4 아파트" + "어서오세요")로 복귀
- 전원을 껐다 켜도 평상시 메시지 유지

## 📌 참고사항 (다빛프로토콜)

### 메시지 타입 구분

| 메시지 타입 | 패킷 형식 | 저장 위치 | 특징 |
|------------|----------|----------|------|
| 실시간 메시지 | ![000/!] | RAM | 덮어쓰기 방식, 전원 OFF 시 사라짐 |
| 페이지 메시지 | ![001/!] | ROM | 영구 저장, 전원 OFF 후에도 유지 |

### 주요 패킷

- `/D0101` : 실시간 메시지 1회 전송
- `/S2010` : 유지시간 설정 (종료 시 빈 화면)
- `/P0000` : 섹션 1
- `/P0001` : 섹션 2
