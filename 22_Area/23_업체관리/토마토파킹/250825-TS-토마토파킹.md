#### 3. 고장 혹은 문의에 대한 상세한 설명 및 해결책
1. 현재 DB500 하드웨어 3.3.2 버전에 96x96 3개를 가로로 이어붙여 사용하고 있습니다. 이미지를 변환하여 표현하려고 하였으나 아래 사진처럼 색상이 제대로 표시되지 않습니다. 문제 해결 방법을 요청드립니다.
	- 7색 이외의 색으로 보여짐
2. DB600 메인보드에 DabitChe 프로그램으로 이미지 업로드 하였습니다. ( 로그에 에러 메시지가 한줄 출력됩니다. )
	1. 이미지 업로드는 되었으나 전광판에 빨간 색 점들이 생겼습니다. 
		- 누끼를 딴 부분에서 색이 잘 못 나오는 것으로 보임, 픽셀단위로 다시 제작
	2. 이미지가 메인보드에 저장되지 않았습니다. ( 실시간 메시지 처럼 작동 ) 
		- pla 파일로 저장하지 않고, bgp 파일로 저장 
3. 이미지 업로드 이후 ASCII 코드 `![001/P0000/Y0008/F0007/E0101/S0320/C2/G0/T0전광판테스트!]` 를 보내봤으나 전광판에 아무런 응답이 없었습니다.
	- 배경이미지가 업로드 되었으니 실시간 메시지(`![000!]`)으로 전송해야 됨
	- 그리고 배경이미지 표시 패킷 (`/B001`) 전송 필수  
4. 현재 Dabitche 프로그램 상 분할 화면으로 이미지를 위/아래로만 가능한데 혹시 위/중간/아래 로 설정이 가능한가요? 6단으로 사용할 경우 3~4단 : 이미지, 1~2단 5~6단 : ASCII 코드 문자로서, 이미지를 고정 한 상태로 문자만을 바꾸는게 가능할까요? 중간에 이미지 넣는게 불가하다면 5~6단 : 이미지(고정), 1~4단 : ASCII 문자(동적) 가능한가요?
	- 가능
