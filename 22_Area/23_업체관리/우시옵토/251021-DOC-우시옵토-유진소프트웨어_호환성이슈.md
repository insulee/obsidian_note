## 유진 소프트웨어의 다빛솔루션 컨트롤러 호환성 이슈


---
#### 문의 내용
유진 소프트웨어로 다빛솔루션 컨트롤러가 검색되지 않음

---
#### 확인 결과

**IP 검색 기능**
- 유진 소프트웨어로 다빛솔루션 컨트롤러(DB400, DB500 등) **IP 검색 불가**
- 반면, 다빛솔루션 소프트웨어로는 유진 컨트롤러(UJM-Controller) **IP 검색 가능**

**유진 소프트웨어와 다빛솔루션 컨트롤러 연결 테스트 및 분석**
- 유진 소프트웨어에서 다빛솔루션 컨트롤러의 **IP와 Port를 직접 입력**하면 문구 표출, 설정 변경 등 실제 사용에 필요한 모든 제어 가능
- 따라서, 유진 소프트웨어가 다빛솔루션 컨트롤러 정보를 읽어서 검색 목록에 표시하지 못할 뿐, **유진 소프트웨어로 다빛솔루션 컨트롤러 제어 가능** 

**결론**
- 다빛솔루션 컨트롤러가 검색이 안되는 문제는, 유진 소프트웨어의 IP 검색 기능 구현이 표준과 차이가 있는 것으로 추정
- **유진 컨트롤러는 다빛솔루션 프로토콜을 기반으로 제작**되어, 통신 및 문구 표출 관련 프로토콜은 동일함
- 프로토콜이 동일하므로, **컨트롤러 교체 시 호환성 문제 없음**

---

#### 선택 사항

**다빛솔루션 소프트웨어 사용**
- 안정적인 IP 검색 및 변경 기능 제공
- 직관적인 인터페이스로 별도 학습 없이 바로 사용
- 다빛솔루션 컨트롤러가 제공하는 다양한 기능을 손쉽게 활용
- 소프트웨어 사용 관련 기술 지원

**유진 소프트웨어 계속 사용**
- IP 검색 기능 대신 수동 IP 입력 방식 사용
- 컨트롤러 IP 주소를 직접 입력하면 모든 기능 제어 가능

---
#### 결론
- 유진과 다빛솔루션의 프로토콜 및 컨트롤러 호환성 확인
- 유진 소프트웨어에서 IP 검색을 제외한 모든 기능을 제어 가능
- 검색 기능이 필요한 경우 다빛솔루션 소프트웨어 사용을 권장
- 유진 소프트웨어를 계속 사용할 경우 IP 수동 입력 방식으로 운용
