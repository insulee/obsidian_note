#### 설정 및 패킷
##### 기본사항 및 설정
- 펌웨어
	- 접이식: `DIBD600T_08C_04R020C_V2.5.3-UM.bin`
	- 고정식: `DIBD500S_08C_02R020C_V2.5.3-UM.bin`
- 다빛채:  `DabitChe_V2.3.2`
	- **(추가)** 색상깜박이기 > 유지시간 `1`로 설정가능
- 다빛채 > 보드 기능 설정 창 > 4. BH1(TTL/RS485) Function: `8Pin Input(BIT BGSch)`
- 다빛채에서 표시파일목록을 배경 이미지로 전송
	- BH1 8핀 입력 접점 신호: 최대 8개의 이미지 파일 중 8개까지 선택 가능
	- 배경 스케줄 목록 작성 프로토콜: 최대 64개의 이미지 파일 중 10개까지 선택 가능
	- 선택된 표출 목록은 RAM에 임시 저장되므로, 전원 리셋 시 이전 설정이 초기화
- 페이지 메시지 갯수: `1`
- 페이지 메시지 18.문자: ` `(띄어쓰기 한칸)
- [ ] **실시간문구 설정: 효과 개별표출 or 효과 동시표출**
	- 실제 .bgp를 넣고 둘의 차이가 있는지 시나리오별로 확인

##### 패킷
- 이미지전송 패킷: `![0020abc def ghi!]`
	- `abc def ghi`: 전광판에 표시할 배경 이미지 번호
	- `![0020001 009 015 030!]`: 배경 이미지 목록 중에서 1번, 9번, 15번, 30번을 스케줄에 등록
- 속도설정 패킷: `![0054 ab!]`
	- `ab`: 전광판에 표시할 배경 이미지 속도
	- `![0054 99!]` => 99(표출 속도가 가장 느림)
	- `![0054 01!]` => 1(표출 속도가 가장 빠름)
	- `![0054 00!]` => 표출 속도 설정 값을 적용하지 않고, 기존 스케줄의 표출 속도를 적용
- 밝기설정 패킷: `![0050ab!]`
	- `ab`: 전광판에 표시할 배경 이미지 밝기
	- `![005099!]` => 99(가장 밝음)
	- `![005001!]` => 1(가장 어두움) 
- 점멸횟수 패킷: `![0055 ab!]`
	- `ab`: 색상깜빡이기 적용시 점멸 횟수 (1회~20회)
		- 기본값은 8회로, `00` 또는 `21` 이상의 숫자는 8회로 설정
	- `![0055 20!]`: 점멸효과 20회
	- `![0055 05!]`: 점멸효과 5회
	- `![0055 00!]`: 점멸효과 8회
	- `![0055 22!]`: 점멸효과 8회
	- 응답패킷
		- `![00550!]` : 설정 성공
		- `![0055F!]` : 설정 실패
