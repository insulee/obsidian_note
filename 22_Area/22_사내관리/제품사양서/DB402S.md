## **DB402S 제품 사양서**

### **1. 제품 개요**

DB402S은 다양한 산업 환경에서 LED 전광판을 제어할 수 있는 **고성능 컨트롤러**입니다.  
다양한 통신 방식을 지원하며, 외부 시스템과의 유연한 연동이 가능합니다.

- **모델명:** DB402S
- **제조사:** 다빛솔루션
- **주요 기능:** LED 전광판 시스템 제어
- **통신 방식:** RS232, TTL, RS485
- **영상신호 출력 포트:** HUB75(16핀)
- **표시 색상:** 7색
- **최대 표시 모듈:** 2단 16열 / 1단 32열

---

### **2. 주요 특징**

1) 멀티 인터페이스 지원
	- RS-232, TTL, RS-485
2) 고성능 듀얼 코어 CPU
	- 멀티 통신 동시 지원
	- 신속한 데이터 처리 가능
3) 검증된 프로토콜 및 전용 프로그램 제공
	- DABIT 프로토콜:
	    - 다빛솔루션의 독자적인 프로토콜로, 전광판 시스템 연동을 위한 간결하고 효율적인 패킷 구조를 제공하여 안정적인 데이터 송수신 및 제어 지원
	- DBPS(DABIT 프로토콜 시뮬레이터):
	    - 실제 환경과 동일한 조건에서 패킷을 사전 전송하여 출력 결과를 미리 확인하고 데이터 연동 과정의 오류를 최소화 
	- 다빛채:
	    - 텍스트, 그래픽, 애니메이션을 편집 및 저장한 후, 컨트롤러에 업로드하여 전광판에 직접 표출 가능
4) 다양한 표시 기능
	- 텍스트, 그래픽, 애니메이션 저장 및 출력 지원
	- 40여 가지 화면 효과, 최대 220자 연속 표시 가능
5) 산업용 환경에 최적화
	- 15년 이상 다양한 산업에서 사용된 검증된 컨트롤러
	- 2년 AS 보장

### **3. 제품 사양**

| 항목             | 사양                                                    |
| -------------- | ----------------------------------------------------- |
| **전원 공급**      | DC 5V, 350mA                                          |
| **최대 표시 모듈 수** | 2단 16열 / 1단 32열                                       |
| **지원 색상**      | 7색 (Red, Green, Yellow, Blue, Magenta, Cyan, White)   |
| **CPU**        | ESPRESSIF Xtensa® dual-core 32-bit LX7 microprocessor |
| **메모리**        | ROM: 384 KB, SRAM: 512 KB, Flash memory: 8 MB         |
| **통신 방식**      | RS232, TTL, RS485                                     |
| **사용 환경**      | -40℃ ~ 85℃                                            |
| **소비 전력**      | 350mA (DC 5V)                                         |
