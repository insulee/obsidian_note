# RS232 GPS 데이터 시트

## 개요

이 GPS 수신기는 RS232 인터페이스를 통해 작동하며, 표준 **NMEA 0183** 프로토콜을 지원합니다. 수신기는 GPS/QZSS 및 WAAS 위성 데이터를 처리하며, 다양한 응용 분야에서 활용할 수 있는 저렴하고 신뢰성 높은 솔루션입니다.

- **지원 메시지**: 수신기에서 출력하는 NMEA 메시지는 다음과 같습니다:
	- **GGA**: 위치 고정 정보
	- **GLL**: 위치 정보
	- **GSA**: GPS 위성 사용 상태 및 정확도
	- **GSV**: 가시 위성 정보
	- **RMC**: 위치, 속도, 시간 정보
	- **VTG**: 지면 속도 및 이동 방향
- **수신 성능**
    - **지원 위성**: GPS, QZSS(일본 위성 시스템), WAAS(미국 정밀 보정 시스템) 데이터를 수신할 수 있습니다.
    - **채널 수**: 66개의 채널을 지원하며, 여러 위성 데이터를 동시에 처리할 수 있습니다.
    - **위치 정확도**: WAAS 지원을 통해 개방된 실외 환경에서는 **2미터의 위치 정확도**를 제공합니다.
    - **최대 업데이트 속도**: GPS 데이터를 초당 최대 10회(10Hz)까지 업데이트할 수 있으며, 기본 업데이트 속도는 1Hz입니다.

### **NMEA 형식**

| **NMEA**        | **설명**                                  | **기본값** |
| --------------- | --------------------------------------- | ------- |
| $Talker ID+GGA  | 시간, 위치 및 위치 고정 관련 데이터                   | Y       |
| $Talker ID+GLL  | 위치 데이터: 위치 고정, 위치 고정 시간, 상태 포함          | Y       |
| $Talker ID+GSA  | GNSS 정확도 및 활성 위성 정보                     | Y       |
| $Talker ID+GSV  | 가시 위성 수, PRN 번호, 고도, 방위각, 신호 대 잡음비(SNR) | Y       |
| $Talker ID+RMC  | 위치, 속도, 시간 정보                           | Y       |
| $Talker ID+VTG  | 지면 방향 및 지상 속도                           | Y       |
| 기타 (ZDA, GNS 등) | 기타 메시지                                  | N       |

#### **(참고) Talker ID**

- GP: GPS
- BD/GB: Beidou
- GA: Galileo
- GL: GLONASS
- GQ: QZSS
- GN: 다중 위성 조합(GPS, GLONASS, Beidou 등 여러 위성 데이터를 결합하여 처리)
