# **DB600 제품 사양서**

</br>

---

### **1. 제품 개요**

DB600은 다양한 산업 환경에서 LED 전광판을 제어할 수 있는 **고성능 컨트롤러**입니다.  
다양한 통신 방식을 지원하며, 외부 시스템과의 유연한 연동이 가능합니다.

- **모델명:** DB600
- **제조사:** 다빛솔루션
- **주요 기능:** LED 전광판 시스템 제어
- **통신 방식:** RS232, TTL, Ethernet, RS485(옵션), WiFi/BLE(옵션)
- **영상신호 출력 포트:** HUB75(16핀)
- **표시 색상:** 7색
- **최대 표시 모듈:** 4단 20열 / 2단 40열 / 8단 10열 / 8단 14열(DB600V)

---

### **2. 주요 특징**
1) 멀티 인터페이스 지원
	- RS-232, TTL, Ethernet, RS-485(옵션), WiFi(옵션), 블루투스(옵션)
	- 다양한 시스템과 유연한 연결 지원
2) 고성능 듀얼 코어 CPU
	- 멀티 통신 동시 지원
	- 신속한 데이터 처리 가능
3) 확장형 통신 및 연동 기능
	- RS-485 통신 추가 가능
	- WiFi/BLE to Serial 컨버터 연동 가능
	- PLC, PC, 임베디드 시스템과 쉽게 통합 가능
4) 검증된 프로토콜 및 전용 프로그램 제공
	- DABIT 프로토콜:
	    - 다빛솔루션의 독자적인 프로토콜로, 전광판 시스템 연동을 위한 간결하고 효율적인 패킷 구조를 제공하여 안정적인 데이터 송수신 및 제어 지원
	- DBPS(DABIT 프로토콜 시뮬레이터):
	    - 실제 환경과 동일한 조건에서 패킷을 사전 전송하여 출력 결과를 미리 확인하고 데이터 연동 과정의 오류를 최소화 
	- 다빛채:
	    - 텍스트, 그래픽, 애니메이션을 편집 및 저장한 후, 컨트롤러에 업로드하여 전광판에 직접 표출 가능
5) 다양한 표시 기능
	- 텍스트, 그래픽, 애니메이션 저장 및 출력 지원
	- 40여 가지 화면 효과, 최대 220자 연속 표시 가능
6) 외부 장치 연동
	- 온습도 센서, 조도 센서, GPS 시간 수신기, 2채널 릴레이 출력 지원
	- 문구 번호 호출 접점보드 연동 가능
7) 산업용 환경에 최적화
	- 15년 이상 다양한 산업에서 사용된 검증된 컨트롤러
	- 2년 AS 보장

### **3. 제품 사양**

| 항목             | 사양                                                  |
| -------------- | --------------------------------------------------- |
| **전원 공급**      | DC 5V, 350mA                                        |
| **최대 표시 모듈 수** | 4단 20열 / 2단 40열 / 8단 10열 / 8단 14열(DB600V)           |
| **지원 색상**      | 7색 (Red, Green, Yellow, Blue, Magenta, Cyan, White) |
| **CPU**        | NXP ARM Cortex-M4 Dual-core 180MHz                  |
| **메모리**        | Flash Memory: 2 MB, SRAM: 264KB                     |
| **통신 방식**      | RS232, TTL, Ethernet(10/100Mbps)                    |
| **시리얼 통신 속도**  | 기본 115,200 bps (9,600~921,600 bps)                  |
| **네트워크 설정**    | 기본 IP: 192.168.0.201 / 포트: 5000                     |
| **사용 환경**      | -40℃ ~ 85℃                                          |
| **소비 전력**      | 350mA (DC 5V)                                       |

### **4. 제품 포트 구성**

| **포트**                   | **설명**                                       |
| ------------------------ | -------------------------------------------- |
| **P1**                   | Ethernet 통신 포트                               |
| **P2**                   | USB C 포트                                     |
| **J1**                   | DC5V 전원 입력                                   |
| **J2**                   | RS-232 통신 포트 (5V, TX, RX, GND)               |
| **J3**                   | TTL 통신 포트 (5V, TX, RX, GND)                  |
| **J4**                   | 2채널 릴레이 출력,  온습도 센서, 조도 센서, GPS 시간 수신기 연결 포트 |
| **JP3**                  | 전압설정 점퍼핀 (3.3V-RS485, 5V-LAN)                |
| **BH1**                  | RS485보드, 접점신호입력보드 연결                         |
| **BH3, BH4<br>BH5, BH6** | HUB75 16핀 영상 신호 출력 포트                        |
| **LED(PWR)**             | 전원 LED (적색, 항상 켜짐)                           |
| **LED(S1)**              | CPU 관련 LED (녹색, 500ms간격 점멸)                  |
| **LED(S2)**              | 화면 표출 관련 (녹색, 500ms간격 점멸)                    |
