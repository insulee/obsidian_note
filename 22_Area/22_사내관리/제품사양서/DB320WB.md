## **DB320WB 제품 사양서**


## **1. 제품 개요**

DB320WB는 **WiFi, BLE 통신을 시리얼(RS232, TTL) 통신으로 전달하는 WifI/BLE to WiFi 컨버터**입니다. WifI, BLE를 통해 전송된 데이터를 시리얼 장치로 전달할 수 있습니다.

- **모델명:** DB320WB
- **제조사:** 다빛솔루션
- **주요 기능** WiFi, BLE와 시리얼(RS232, TTL) 간의 데이터 전달
- **통신 프로토콜:** TCP(Server/Client), UDP
- **통신 인터페이스:** RS232, TTL, WiFi(IEEE 802.11 b/g/n, 2.4GHz), BLE(Bluetooth Low Energy)
- **WiFi 모드:** AP Mode(기본), Station Mode
- **기본 네트워크 설정:** AP 모드(IP: 192.168.4.1, 포트: 5100)
- **기본 시리얼 설정:** 115,200 bps, N, 8, 1(Stop bit)


---

## **2. 주요 특징**

1. **WiFi 및 Bluetooth 통신 지원**
    - WiFi(IEEE 802.11 b/g/n, 2.4GHz)를 통해 원격 제어 가능
    - BLE 통신 기능을 지원하여 다양한 무선 장치와 연결 가능
    - WiFi/BLE 동시 사용 가능
2. **AP 및 STA 모드 지원**
    - **AP 모드(기본값):** DB320WB가 자체적으로 WiFi 신호를 발생하여 스마트폰, PC 등의 기기와 직접 연결 가능
    - **STA 모드:** 사용자의 공유기 네트워크에 연결하여 네트워크 내 다른 장치들과 데이터 송수신 가능
3. **산업 환경에 최적화된 안정성 제공**
    - -40 ~ 85℃의 넓은 동작 온도 범위
    - 자동 재연결 기능으로 네트워크 장애 발생 시 자동 복구
    - 하드웨어 초기화 버튼 및 원격 리셋 기능 제공
4. **다양한 시리얼 통신 장치 지원**
    - RS232 및 TTL 포트 각각 1개 제공
    - 산업용 PLC, 전광판 컨트롤러, 계측 장비, 자동화 시스템과 연동 가능
5. **손쉬운 설정 및 사용**
    - **다빛솔루션 제공 프로그램**을 통해 네트워크 및 시리얼 설정 가능
    - 다양한 LED 상태 표시 기능으로 **실시간 동작 상태 확인 가능**

## **4. 포트 구성 및 LED 상태 표시**

제품 전면사진

거버파일 사진

| **포트**       | **설명**                          |
| ------------ | ------------------------------- |
| **J1**       | DC5V 전원 입력                      |
| **J2**       | RS-232 통신 포트 (5V, TX, RX, GND)  |
| **J3**       | TTL 통신 포트 (5V, TX, RX, GND)     |
| **J4**       | 2채널 릴레이 출력 포트                   |
| **J5**       | 온습도 센서, 조도 센서, GPS 시간 수신기 연결 포트 |
| **BH1**      | RS485보드, 접점신호입력보드 연결            |
| **LED(PWR)** | 전원 투입 시 적색 점등                   |
| **LED(S1)**  | CPU 정상 동작 시 1초 간격 점멸 (녹색)       |

## **5. 사용 방법**

6. **WiFi를 통한 원격 접속(AP 모드)**
    - DABIT300 WiFi의 AP 모드 기본 설정:
        - **SSID:** `AP-dabitnet`
        - **비밀번호:** `dabitnet`
    - 스마트폰 또는 노트북에서 `AP-dabitnet` WiFi에 연결
    - 다빛채(PC, 모바일) 실행 후 기본 IP(192.168.4.1)로 접속하여 데이터 송수신
7. **STA 모드 설정(공유기 연결)**
    - DABIT300 WiFi를 유무선 공유기에 연결하여 동일 네트워크 내 다른 장치와 통신 가능
    - 설정 변경 방법:
        - **모드:** STA
        - **SSID:** 사용자의 공유기 SSID
        - **Password:** 공유기 비밀번호 입력
        - **IP 설정:** 192.168.0.xxx (공유기 대역에 맞게 설정)
8. **시리얼 통신 장치 연결**
    - TTL 또는 RS232 장치를 연결하여 WiFi 또는 Bluetooth로 데이터 송수신 가능
    - 전광판 컨트롤러(DB500)와 연결하여 원격 제어 가능
9. **하드웨어 초기화 기능**
    - 리셋 버튼을 눌러 공장 출하시 기본 설정으로 복원
    - 초기화 후 기본값:
        - **WiFi:** AP 모드 (SSID: AP-dabitnet, PW: dabitnet)
        - **IP:** 192.168.4.1, 포트 5100
10. **UDP 통신으로 원격 리셋 기능**
    - 네트워크 장애 시, UDP 패킷으로 원격 리셋 가능
    - 패킷: `"++SET++![0041!]"`
    - **UDP 포트:** 5107
