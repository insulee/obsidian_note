## **DABIT300 제품 사양서

## 1. 제품 개요

DABIT300은 이더넷(TCP/IP, UDP) 통신을 시리얼(RS232, TTL) 통신으로 전달하는 LAN to Serial Converter입니다.  네트워크로 전송된 데이터를 시리얼 통신 장치에 전달할 수 있습니다.

- 모델명: DB300E
- 제조사: 다빛솔루션
- 주요 기능: LAN(이더넷)과 시리얼(RS232, TTL) 간의 데이터 전달
- **통신 프로토콜:** TCP(Server/Client), UDP
- 통신 인터페이스: RS232, TTL, Ethernet(10/100Mbps)
- 기본 네트워크 설정: IP: 192.168.0.202, 포트: 5000 (TCP Server)
- 기본 시리얼 설정: 115,200 bps, N, 8, 1(Stop bit)

---

## 2. 주요 특징

1. LAN ↔ Serial 변환 기능 제공
    - TCP/IP 또는 UDP 네트워크를 통해 원격 장치와 시리얼(RS232, TTL) 장치 간 데이터 송수신 가능
    - 산업용 장비, IoT 시스템, 전광판 컨트롤러, POS 시스템 등의 시리얼 장치를 네트워크에 연결
2. 다양한 통신 인터페이스 지원
    - 이더넷: 10/100Mbps, TCP/IP, UDP
    - 시리얼: RS232, TTL
3. 산업 환경에 최적화된 안정성 제공
    - -40 ~ 85℃의 넓은 동작 온도 범위
    - 자동 재연결 기능으로 네트워크 장애 발생 시 자동 복구
    - 하드웨어 초기화 버튼 및 원격 리셋 기능 제공
4. 손쉬운 설정 및 사용
    - 다빛솔루션 제공 프로그램을 통해 네트워크 및 시리얼 설정 가능
    - 다양한 LED 상태 표시 기능으로 실시간 동작 상태 확인 가능

## **4. 포트 구성 및 LED 상태 표시**


제품 전면사진

거버파일 사진

| **포트**           | **설명**                        |
| ---------------- | ----------------------------- |
| **P1**           | RJ45 이더넷 포트 (TCP/IP, UDP)     |
| **J2**           | TTL 통신 포트 (5V, TX, RX, GND)   |
| **J2**           | RS232 통신 포트 (5V, TX, RX, GND) |
| **BH1**          | TTL 연결 포트(DB502+300 세트 연결시)   |
| **LED(PWR)**     | 전원 투입 시 적색 점등                 |
| **LED(S1)**      | CPU 정상 동작 시 1초 간격 점멸 (녹색)     |
| **LED(LAN-주황색)** | LAN 케이블 연결 시 점등               |
| **LED(LAN-녹색)**  | 이더넷 데이터 수신 시 점멸               |
