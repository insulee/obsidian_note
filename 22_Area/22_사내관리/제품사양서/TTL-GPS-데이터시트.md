# TTL GPS 데이터 시트
## **개요**

이 GPS 모듈은 TTL 직렬 통신을 통해 작동하며, 표준 **NMEA 0183** 프로토콜을 지원합니다. 고감도 GPS 수신기로서, 해양 전자 제품, 운송 시스템, IoT 등 다양한 응용 분야에서 활용할 수 있는 경제적이고 신뢰성 높은 솔루션입니다.
- **지원 메시지**: 수신기에서 출력하는 NMEA 메시지는 다음과 같습니다:
	- **GGA**: 위치 고정 정보
	- **GLL**: 위치 정보
	- **GSA**: GPS 위성 사용 상태 및 정확도
	- **GSV**: 가시 위성 정보
	- **RMC**: 위치, 속도, 시간 정보
	- **VTG**: 지면 속도 및 이동 방향

## **NMEA 형식**

| **NMEA 문장** | **설명**                                  |
| ----------- | --------------------------------------- |
| $GPGGA      | 시간, 위치 및 위치 고정 관련 데이터                   |
| $GPGLL      | 위치 데이터: 위치 고정, 위치 고정 시간, 상태 포함          |
| $GPGSA      | GNSS 정확도 및 활성 위성 정보                     |
| $GPGSV      | 가시 위성 수, PRN 번호, 고도, 방위각, 신호 대 잡음비(SNR) |
| $GPRMC      | 위치, 속도, 시간 정보                           |
| $GPVTG      | 지면 방향 및 지상 속도                           |

## **참고 사항**

1. **전원 공급**: 모듈은 **DC 3.6V ~ 5.5V** 전압에서 작동하며, 일반적인 동작 전압은 5.0V입니다.
2. **통신 인터페이스**: **TTL 직렬 데이터 통신**을 사용하며, TXD/RXD 핀을 통해 데이터 송수신을 처리합니다.
3. **설정 저장**: 4M 플래시 메모리를 사용하여 설정을 영구적으로 저장할 수 있습니다.
4. **환경 조건**: IP67 방수 등급으로 야외 환경에서도 사용 가능하며, -40℃ ~ +85℃에서 안정적으로 작동합니다.
