#### DB500S + DB300E TTL 연결 테스트
- **DB500S와 DB300E를 연결한다면 RS232로 연결할 것을 권장**
- **TTL로 연결하겠다 한다면 저항을 바꾼 DB300E로 안내**
- 저항 바꾸기 전/후 구분해서 관리
%%
- 테스트 환경
	- 컨트롤러: DB500S (펌웨어 v3.3.5)
	- 컨버터: DB300E (펌웨어 v2.0.4)
	- 연결방식: 4핀케이블로 TTL 포트 연결
- 이슈 사항
	- DB500S(V13) + DB300E(V06) TTL 포트로 연결시 일부 제품 연결 오류 발생
		- DB500S v3.3.5이외의 버전에서도 연결오류(V2.4.1. V2.5.3 등)
		- 모든 조합에서 RS232 연결은 정상 작동
		- DB500S(V13) TTL to USB는 정상 연결 확인
	- DB600T, DB502는 모든 버전에서 DB300E와 정상 연결
	- 특정 제품은 V06 만 연결이 안되는 문제 (표본 확인 필요)
- **DB300E 저항 변경 후 해결**
