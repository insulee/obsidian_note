#### 문제 개요 
- **성화기술산업 보고:** USB를 통한 컨트롤러 설정 시 USB가 인식되지 않음.
- **드래곤사이니지 보고:** USB를 꽂았을 때 센서 데이터(TXT 파일)가 USB에 저장되지 않음.
- **공통점:** 두 업체 모두 SanDisk Cruzer Blade 제품을 사용했으며, 해당 제품으로 테스트 결과 동일한 인식 불가 현상이 재현
- 성화기술산업과 드래곤사이니지의 주문형 펌웨어는 `DIBD502S_08C_02R020C_V3.3.0.bin`으로 출고전 테스트는 **Axxen사의 USB로 정상 표출 확인 후 출고**

#### 다빛솔루션 컨트롤러별 USB 인식 테스트 결과

- **DB500S / DB600T:**
    - **V3.2.3 펌웨어:** 모든 USB 인식
    - **V3.2.8 펌웨어:** 모든 USB 인식 불가능
- **DB502S:**
    - **V2.4.3 미세먼지 펌웨어:** 모든 USB 인식
    - **V3.3.0 펌웨어 (미세먼지/구급차):** 
        - Axxen사 USB는 정상 인식
        - TK사 USB는 가끔 인식 안됨
        - SanDisk사 USB는 인식 불가능
