- 일반사항
	- [x] 상단에 팝업 제목을 DabitNet -> dbNet으로 변경
	- [ ] COMM > UDP > Network Target > **All** 이 기본으로

	- [x] DabitNet과 동시에 TCP/IP 찾기 안됨 
		- `​[24/12/20 16:27:01] [ERROR!] IP: 255.255.255.255, PORT: 5108열기에 실패했습니다.
		- 기존 시뮬레이터도 마찬가지
		- 포트 5107에서 검색 실패
	- [x] 기존 프로그램(V2.3.0)에서 TCP/IP 연결 후 연결 시도하면 포트 열리지 않음 ✅ 2025-01-06

	- [ ] Controller Client Mode > 체크 안됨
	- dbNet Add 적용 오류
		- [ ] dbNet > Controller Client Mode 체크 > Add > Server TCP/IP에 체크 안됨
		- [ ] dbNet > Controller Client Mode 미체크 > Add > Client TCP/IP에 체크 안됨
		- IP Address, Port는 적용
	- [x] Controller Client Mode > 체크 안되면 아래 IP / Port 비활성화
