# MQTT 프로젝트 설정 가이드

## 1. Mosquitto 브로커 설정

```bash
mosquitto -c "C:\Program Files\mosquitto\mosquitto.conf" -v
```

## 2. Python 환경 설정

### 가상환경 생성 및 활성화

```bash
# 가상환경 생성 (최초 1회만 실행)
python -m venv venv

# 가상환경 활성화
# PowerShell에서:
.\venv\Scripts\Activate.ps1
# 또는 CMD(명령 프롬프트)에서:
# venv\Scripts\activate.bat
```

### 필요 패키지 설치

```bash
pip install paho-mqtt
pip install Pillow
```

## 3. 프로젝트 실행

```bash
python main.py
```

## 참고 사항

### Python 버전 확인

```bash
python --version
```

### 문제 해결

PowerShell에서 스크립트 실행이 차단된다면, 관리자 권한으로 PowerShell을 열어 다음 명령을 실행하세요:

```bash
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```
