- 하드웨어 연결
	- SERVER -- DB300eM --(RS232)-- 전광판 컨트롤러 -- LED모듈
- 소프트웨어 연결: RoaDee Image Uploader.exe
- 기본설정
	1. MQTT 연결: 아래의 기본 브로커 정보를 입력하고 연결
		- 브로커: 192.168.1.80
		- 포트: 1883
		- ID: dabit
		- PW: dabit
	2. 화면 설정: 아래의 화면 설정 정보를 입력하고 전송
		- 가로 모듈: 2
		- 세로 모듈: 6
		- 양면 여부: 단면
		- 배열 방향: 가로형
	3. 밝기 설정
	4. 현재 시간: 서버시간 동기화 버튼으로 시간 동기화
	5. 로그 > 전광판 설정 읽기 버튼으로 현재 설정 확인
- 이미지 업로드 및 표출 순서: 아래의 순서로 이미지를 업로드하면 전광판에 업로드 된 이미지가 순서대로 표출
	1. 파일 선택
		- 이미지 파일을 선택 후 열기
		- 최대 10개까지 업로드 가능
	2. 청크 크기
		- 한번에 보낼 수 있는 청크의 크기
		- 1024, 2048, 4096byte 선택 가능
	3. 응답 대기시간
		- 응답을 받기 위해 최대 얼마만큼의 시간을 기다릴지를 정해 놓은 시간(초)
		- 최대 10초까지 설정 가능
	4. 전송 후 자동 배포 (deploy)
		- 체크(x)하면 이미지 전송 후 전광판에 바로 표출
		- 체크 안하고 프로그램 하단에 수동 커밋을 누르면 전광판에 표출 가능
	5. 효과
		- 현재는 정지효과(none)와 왼쪽으로 이동효과(move_left)만 구현
		- 다른 효과는 정지효과로 표출
	6. 속도
		- 효과가 나타나는 속도로 숫자가 작을수록 빠르게 효과가 나타남
	7. 지연
		- 효과가 나타나고 유지되는 시간(초)
	8. 전송 시작
- 스케줄 관리
	- 명령
		- 목록 조회 > 실행 버튼을 누르면 업로드한 이미지의 목록 생성
		- 모두 삭제 > 실행 버튼을 누르면 업로드한 이미지의 목록 삭제
