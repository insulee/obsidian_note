# 카카오톡 챗봇 시나리오 및 대화 플로우 설계

> AS 기술문의 대응을 위한 체계적인 시나리오 및 대화 흐름 설계

---

## 시나리오 설계 목표

### 주요 목표
- **직관적이고 단계적인 문제 해결 과정** 제공
- **멀티턴 대화**를 통한 정확한 문제 진단
- **사용자 친화적인 UI/UX** 경험 제공
- **상담원 연결**까지의 매끄러운 에스컬레이션

---

## 주요 시나리오 카테고리

### 1. 제품 연결 및 설정 문제
- 네트워크 연결 이슈
- 블루투스 페어링 문제
- WiFi 설정 문제
- 초기 설정 가이드

### 2. 기기 오작동 및 고장
- 전원 관련 문제
- 디스플레이 이상
- 소리/음성 문제
- 하드웨어 고장 진단

### 3. 소프트웨어 및 펌웨어
- 업데이트 관련 문제
- 앱 연동 이슈
- 설정 변경 방법
- 리셋 및 초기화

### 4. 사용법 및 기능 문의
- 기본 사용법 안내
- 고급 기능 활용
- 매뉴얼 및 가이드 제공
- FAQ 대응

### 5. AS 접수 및 문의
- 보증기간 확인
- AS 접수 안내
- 수리 진행 상황 조회
- 상담원 연결

---

## 대화 플로우 기본 구조

### 기본 패턴
```
사용자 문의 입력
    ↓
의도 파악 및 분류
    ↓
초기 질문/확인사항
    ↓
단계별 해결 시도
    ↓
├─ 해결 완료 → 만족도 조사
├─ 추가 도움 필요 → 심화 진단
└─ 해결 불가 → 상담원 연결
```

### 멀티턴 대화 예시

#### 네트워크 연결 문제 시나리오

1. **사용자**: "WiFi 연결이 안 돼요"
2. **챗봇**: "WiFi 연결 문제를 도와드리겠습니다. 현재 다른 기기(스마트폰, 노트북 등)는 같은 WiFi에 정상적으로 연결되나요?"
   - 버튼: [네, 잘 돼요] / [아니요, 안 돼요]

3-A. **[네, 잘 돼요] 선택시**:
   - **챗봇**: "다른 기기는 정상이군요. 제품의 WiFi 설정을 확인해보겠습니다. 제품 화면에 WiFi 아이콘이 표시되나요?"
   - 버튼: [표시됨] / [표시 안됨] / [화면이 없어요]

3-B. **[아니요, 안 돼요] 선택시**:
   - **챗봇**: "인터넷 자체에 문제가 있을 수 있습니다. 공유기(WiFi 라우터)의 전원을 5초간 끈 후 다시 켜보세요. 그 후 다시 시도해주세요."
   - 버튼: [시도해볼게요] / [해봤는데 안 돼요]

## 시나리오별 세부 설계

### 기술문의 시나리오 템플릿

**시나리오명**: [문제 유형]
**예상 키워드**: [사용자가 입력할 가능성이 높은 단어들]
**해결 목표**: [이 시나리오가 달성하고자 하는 결과]

#### 단계별 대화 흐름:
1. **문제 확인 단계**
   - 현상 파악 질문
   - 기본 상황 체크

2. **진단 단계**
   - 단계별 점검 사항
   - 분기별 대응 방안

3. **해결 시도 단계**
   - 구체적 해결 방법 안내
   - 결과 확인

4. **완료 또는 에스컬레이션**
   - 해결 완료 시: 만족도 조사
   - 미해결 시: 상담원 연결

## 사용자 경험 (UX) 고려사항

### 대화 톤앤매너
- **친근하고 도움이 되는 톤** 유지
- **전문 용어 최소화**, 쉬운 설명 위주
- **단계별 안내**로 사용자 부담 최소화

### 버튼 및 Quick Reply 활용
- **Yes/No 질문**은 반드시 버튼으로 제공
- **선택지가 3개 이상**일 때는 Quick Reply 활용
- **"이전 단계로"** 버튼 항상 제공

### 예외 상황 처리
- **이해하지 못한 입력**에 대한 명확한 안내
- **반복 질문**에 대한 적절한 대응
- **에스컬레이션 경로** 항상 열어두기

## 외부 시스템 연동

### 필요한 연동
- **제품 DB**: 시리얼 번호 기반 제품 정보 조회
- **AS 시스템**: 접수 및 진행 상황 조회
- **지식 베이스**: RAG 시스템 연동
- **상담원 시스템**: 실시간 상담 연결

---

## 성과 측정 지표

### 주요 KPI
- **문제 해결률**: 챗봇만으로 해결된 비율
- **평균 대화 턴 수**: 문제 해결까지의 대화 횟수
- **사용자 만족도**: 5점 척도 평가
- **상담원 연결률**: 에스컬레이션 비율

### 개선 지표
- **미처리 질문 유형** 분석
- **대화 이탈 지점** 파악
- **자주 묻는 질문** 업데이트 필요성
