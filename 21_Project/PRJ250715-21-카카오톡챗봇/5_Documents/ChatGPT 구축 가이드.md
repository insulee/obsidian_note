# Kakao i Open Builder 기반 기술문의 챗봇 구축 종합 가이드

> ChatGPT가 제공하는 카카오톡 AS 기술문의 챗봇 구축을 위한 완전한 가이드

---

## 개요

**고객 AS 기술문의 대응용 챗봇 구축 개요:**  
Kakao i Open Builder를 활용하면 **카카오톡 채널** 상에서 동작하는 AI 챗봇을 개발할 수 있습니다. 본 가이드는 카카오톡 챗봇을 처음 기획하는 단계부터 시나리오 설계, 학습 데이터 구축, 운영 및 유지보수, 그리고 사용자 유입과 홍보 전략까지 폭넓게 다룹니다. 

특히 FAQ 수준을 넘어 **시나리오 기반의 단계적 문제 해결**을 제공하는 챗봇을 목표로 하므로, **멀티턴 대화 흐름 설계**와 **지속적인 개선**에 중점을 두었습니다.

## 1. Kakao i Open Builder로 챗봇 구축 시작하기

### 1-1. 카카오톡 채널 및 개발자 계정 준비

챗봇을 만들기 전에 **카카오톡 채널**을 개설해야 합니다. 기업이나 브랜드 계정으로 [카카오 비즈니스](https://business.kakao.com/)에 가입한 뒤 카카오톡 채널을 생성합니다. 

**주요 설정 사항:**
- **프로필 사진, 채널 이름, 검색용 ID, 소개글, 카테고리** 설정
- _채널 이름은 변경이 매우 어려우므로 신중하게 결정_
- **프로필 공개 및 검색 허용** 반드시 **ON**으로 설정
- **고유 URL** 확인 및 기록 (예: `https://pf.kakao.com/_abcd1234`)

### 1-2. 챗봇 관리자센터 OBT 신청

카카오톡 챗봇 개발을 위해서는 **Kakao i Open Builder** 사용 신청(OBT 신청)이 필요합니다.

**신청 절차:**
- 카카오 계정으로 로그인하여 OBT 신청서 작성
- 카카오톡 채널 URL 입력
- **개인 계정 신청 권장** (기업 계정은 추가 서류 요구)
- **승인 기간**: 보통 3~5일 내외
- 결과는 카카오 계정 이메일로 안내

### 1-3. 챗봇 생성 및 기본 설정

OBT 승인 완료 후 **챗봇 관리자센터**(i.kakao.com)에서 챗봇 생성:

1. **[서비스/도구] - [챗봇] - [내 챗봇]** 메뉴
2. **「+ 봇 만들기」** → **카카오톡 챗봇** 선택
3. 챗봇 이름 입력 (추후 변경 가능)
4. **설정 메뉴**에서 환영인사, 봇 프로필 이미지 설정

### 1-4. 카카오톡 채널과 챗봇 연동

생성한 챗봇을 실제 카카오톡 채널과 연결:

1. **[설정] - [카카오톡 채널 연결]** 메뉴
2. **운영 채널 연결** 옵션에서 카카오톡 채널 선택
3. **주의사항**: 챗봇은 채널과 1:1로 연결됨
4. **채팅방 메뉴** 기능이 켜져 있으면 챗봇 연결 제한

### 1-5. 개발/운영 채널 분리와 테스트

**권장 사항:**
- **개발용 채널**(비공개) 별도 생성하여 테스트
- 충분한 테스트 후 **운영 채널**에 연결
- **미리보기 기능** 또는 실제 카카오톡 앱에서 검증
- **사용성 테스트**: 동료/내부 직원 대상 베타 테스트

### 1-6. (선택 사항) 상담원 연결 설정

복잡한 이슈나 불만 접수를 위한 **상담원 연결** 기능:

- **[상세설정] - [채널 채팅방]** → **"상담사 연결버튼 사용"** ON
- 채팅창 하단에 **"상담원 연결"** 버튼 표시
- **1:1 채팅** 기능 활성화 필요

## 2. 기술문의 시나리오 설계 방법

### 2-1. 시나리오와 블록 개념 이해

**핵심 개념:**
- **시나리오(Scenario)**: 챗봇이 제공하는 하나의 서비스 단위
- **블록(Block)**: 사용자 의도(Intent)에 반응하는 가장 작은 응답 단위
- **시나리오 구성**: 여러 블록들이 모여 하나의 시나리오 구성

**예시 구조:**
```
AS 기술지원 챗봇
├── 네트워크 문제 시나리오
│   ├── 연결 확인 블록
│   ├── 설정 점검 블록
│   └── 해결 완료 블록
├── 계정 문제 시나리오
└── 기기 오류 시나리오
```

### 2-2. 기술문의 유형 분류

**주요 문의 카테고리 예시:**
- "전원이 안 켜져요"
- "네트워크 연결 문제"
- "오류 메시지 발생"
- "설정 방법 문의"
- "제품 사용법 안내"

**시나리오 설계 원칙:**
- 각 카테고리별 **세부 해결 플로우** 구상
- **"사용자 질문 → 챗봇 응답 → 추가 질문/분기 → 해결 또는 종료"** 흐름
- **시나리오별 예상 발화 패턴** 정의
