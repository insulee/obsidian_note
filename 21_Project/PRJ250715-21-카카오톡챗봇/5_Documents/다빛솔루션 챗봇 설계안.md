## 챗봇의 목적

1. 반복적인 기술문의(제품 스펙, 매뉴얼, 설치 방법)를 자동화하여 **업무 부담 감소 및 비용 절감**
2. 고객이 문의하는 내용을 **신속하게 분류**하여, 간단한 문제는 즉시 해결하고 복잡한 문제는 **AS/기술지원팀으로 빠르게 전달**
3. LLM(GPT/Gemini 등) 연동을 통해 **예외 상황에도 최대한 대응** 가능하도록 보강
4. 고객 경험 관리: 챗봇이 해결하지 못하는 부분은 **명확한 Fallback(상담사/AS 연결)** 제공
5. 최신 제품 정보, 매뉴얼, 펌웨어, 프로토콜 안내를 고객이 **24시간 언제든지 확인 가능**

## 블록 설계 (예시)

### 1. 제품 문의 블록

- 컨트롤러 종류 (DB300E, DB502S, DB600 등) → 상세 스펙/매뉴얼 제공
- 모듈 문의 (P10, 풀컬러/단색, HUB75 등) → 호환성 안내
- 옵션보드 (변환보드, 릴레이, GPS, 센서) → 용도 설명 및 연결 방법
- 통신방식 (LAN, RS232, RS485, Wi-Fi, BLE) → 연결 가이드

---

### 2. 구매 문의 블록

- **사업자 구매**: 사업자등록증 메일 첨부 안내 → 견적서 발송 절차
- **소량 구매**: 네이버 스마트스토어 링크 안내
- **견적 문의**: 제품 수량, 옵션 선택 입력 → 담당자 메일 안내

---

### 3. 사용 방법 문의 블록

- DabitDocs(매뉴얼) 검색 연결
- 설치/설정: IP설정, Wi-Fi/BLE 연결, 초기화 방법
- 펌웨어 업데이트: 최신 버전 다운로드 링크 제공
- 보드 설정: 포트별 기능 설명, 예시 이미지 제공
- 모듈 연결법: 모듈 매핑 방법, HUB75 케이블 안내
- FAQ: “글자가 안나온다”, “화면이 깜빡인다” 등 자주 묻는 문제

---

### 4. 프로토콜/개발 블록

- ASCII 프로토콜 기본 구조
- 예제 메시지 (`$메시지형식,데이터1,데이터2,...`)
- 자주 발생하는 오류/예외 케이스
- 개발자 전용 문서/샘플 코드 제공
- 담당자 이메일: [dabit@dabitsol.com](mailto:dabit@dabitsol.com)
    

---

### 5. 기술문의/AS 블록

- “~가 안된다” 유형 입력 → 키워드 매칭 (예: 통신, 설치, 전원, 화면 등)
- 매뉴얼/FAQ 자동 제공
- 해결 불가 시 → AS/기술지원 접수 안내
    - 택배 접수 주소 안내
    - 수리 절차 / 테스트 진행 안내
- 최종 Fallback: “담당자가 직접 연락드리겠습니다.”

---

### 6. 일반 문의 블록

- 영업/마케팅 문의: 협업 제안 접수
- 회사 위치, 전화번호, 이메일 안내
- 소프트웨어 다운로드: DabitDocs → 다운로드 페이지 연결
- 매뉴얼 링크: DabitDocs 바로가기
- 제품 Spec: 홈페이지 링크
