## DB502V10
#### 사진 및 설명

| 항목  | 설명                               |
| --- | -------------------------------- |
| P1  | USB C 포트                         |

| BH4 | 영상 신호 출력 포트 (16핀, 2단)            |
| JP1 | 하드웨어 리셋용 점퍼                      |
| JP3 | BH1 포트 통신 방식 선택용 점퍼 (LAN, RS485) |

## 컨트롤러 설정 및 IP 정보

#### DB502V10 컨트롤러 설정
- 전광판 크기
	- 세로: `1`
	- 가로: `18`
	- 픽셀당 비트수: `3BPP(8Color)`
- 표출신호: `16D-P16D1S21`
	- Color Order: `GBR`
- 특수기능 > 페이지문구 페이지 개수 등록: `1`
- 페이지문구> 
	- 1\. 페이지번호: `1` 
	- 18\. 문자: ` `(띄어쓰기 1칸)
