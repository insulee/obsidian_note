## Jira

- 프로젝트 일정 및 할일 관리
- 주간업무보고 - AS/기술문의, 완료된 업무, 할일 등
#### 초기세팅

- 프로젝트 만들기 → 프로젝트 관리
- 프로젝트 메인화면 구성
    - 요약
    - 타임라인 (기본값으로 설정) ***
    - 목록***
        - 담당프로젝트 및 할일을 나열
        - 설명, 담당자, 기한 등 기입
    - 보드
    - 이슈
    - 캘린더
    - 첨부파일
    - 페이지
- 이슈 유형 및 컨텍스트 필드
    - 에픽: 담당자, Start date, 기한
    - Task: 담당자, Start date, 기한, 우선 순위
    - Sub-task: 담당자, Start date, 기한, 우선 순위

#### Issue별 내용

- 프로젝트: 브랜드마케팅팀
- Epic: 브랜드마케팅팀의 담당 프로젝트
    - 프로젝트의 범위가 크고, 할일이 많다면 별도의 프로젝트 생성
    - ex. DBPS 테스트 프로젝트
- Task: 프로젝트의 할일/기한 등을 정해서 한번에 보기 쉽게 정리
#### 브랜드마케팅팀의 담당 프로젝트

- 상시 프로젝트
    - AS/기술문의
    - 주간업무 - 250203-250207
    - DABIT DOCS
    - 네이버 스마트스토어 운영
    - 소셜미디어 운영
- 완료가 있는 프로젝트
	- 홈페이지 수정
	- 네이버 스마트스토어 제작
- 프로젝트/주간업무 완료시 Archived prj/issue로 이동
    

## Obsidian

- 일간노트에 할 일, 업체 연락 온 사항 등을 메모
- 문서 작성 및 저장
    - 작성된 문서는 Share Note 플러그인으로 링크 생성해 Jira 설명에 추가
    - 프로젝트 관련 문서는 21. Project에 폴더를 만들어 작성
    - AS/기술문의 관련 문서는 23. Resource > 하자처리(A_S) 에 작성
    - DABIT DOCS 매뉴얼은 DABIT DOCS Vault에 작성
