		- 문서화 과정에서 복잡한 개념이나 기술적인 내용을 쉽게 설명하기 위해 어떤 기법을 사용하시나요?
		- 문서화된 내용을 지속적으로 유지 관리하기 위해 어떤 방식을 사용하시나요?
		- 문서 작성 시 독자의 이해도를 높이기 위해 어떤 방법을 사용하시나요?
