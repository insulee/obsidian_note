### 1. 개요

- ROADEE의 전광판 정보에 해당하는 프로토콜이 업데이트 됨에 따라, 주요 변경 사항은 다음과 같습니다. 
	1. `2.RTE058.2: Message List` 삭제 
	2. 이미지 파일 전송 (`2.RTE058.4`) 기능 추가
	3. 이미지 스케줄 관리 (`2.RTE058.5`) 기능 추가 

### 2. 주요 변경 사항

#### 2.1. `2.RTE058.2: Message List` 삭제 및 새로운 방식 적용

- 기존 버전 (`2.RTE058.2: Message List`)
	- 메시지를 실시간 문구(0) 및 페이지 문구(1~n) 로 구분하여 개별 관리
	- 섹션별로 메시지 표시 방식(폰트 크기, 색상, 이동 효과 등)을 설정
	- 특정 위치에 표시하기 위한 좌표 기반 설정
	- 메시지 전송과 이미지 파일 전송을 동시에 구현하는데 시스템 자원의 한계와 많은 시간이 소요될 것으로 예상

- 새로운 방식 적용
	- 문구 기반 메시지 전송 방식 제거하여 메시지를 실시간으로 입력하여 전송하지 않음
	- ROADEE에는 이미지 표출이 필요하다는 의견에, `이미지 파일 전송` 기능을 추가하여 표출하고자 하는 **텍스트와 이미지(픽토그램)를 픽셀 기반 이미지로 제작하여 전송**
	- `이미지 스케줄 관리` 기능을 추가하여 **사전에 등록된 이미지 스케줄에서 사용자가 원하는 시점에 특정 이미지를 호출하여 표출**
	- 기존 버전 (`2.RTE058.3 Display Board Info(Set)`)에 페이지 문구 개수 설정, 페이지 문구 삭제와 같은 페이지 문구관련 MO를 삭제


#### 2.2. `2.RTE058.4: 이미지 파일 전송` 기능 추가

- 기능 개요
	- 픽셀 기반으로 제작한 이미지 파일 전송 가능
	- Base64로 인코딩된 이미지 파일 데이터를 최대 1500바이트 단위로 분할 전송
	- `이미지 파일 전송 상태`, `이미지 파일명`, `전송 정보` 등을 포함
- 운용 방식
	- 사용자는 전광판에 표시할 내용을 미리 픽셀 기반의 이미지 파일로 제작
	- 해당 이미지를 서버에서 전광판 제어기(컨트롤러)로 전송하여 컨트롤러에 스케줄 형태로 저장

#### 2.3. `2.RTE058.5: 이미지 스케줄 관리` 기능 추가

- 기능 개요
	- `이미지 스케줄 관리` 기능을 추가하여 특정 시점에 원하는 이미지 표출 가능
	- `이미지 효과`, `이미지 효과 파라미터` 기능으로 정지, 깜빡이기 이동 등의 애니메이션 적용과 반복 간격, 유지시간, 속도 등의 속성 설정 가능
- 운용 방식
	- 사용자는 `이미지 파일 전송` 기능을 통해 컨트롤러에 스케줄 형태로 저장된 여러 개의 이미지를, 특정 상황에서 특정 이미지를 호출하여 표출 가능


### 3. 주요 변경 사항 요약

| 구분         | 기존 버전                              | 업데이트 버전                     | 주요 변경점                        |
| ---------- | ---------------------------------- | --------------------------- | ----------------------------- |
| 메시지 전송 방식  | `Message List`를 통한 실시간 및 페이지 문구 설정 | `이미지 파일 전송` 방식으로 변경         | 메시지 기반 전송 방식에서 이미지 기반 전송으로 변경 |
| 이미지 파일 전송  | 없음                                 | `2.RTE058.4: 이미지 파일 전송` 추가  | 컨트롤러에 미리 제작한 이미지 업로드 가능       |
| 이미지 스케줄 관리 | 없음                                 | `2.RTE058.5: 이미지 스케줄 관리` 추가 | 등록된 이미지 중 특정 시점에 원하는 이미지를 표출  |


### 4. 결론 및 적용 가이드

- 기존 메시지 전송 방식(`Message List`)이 제거되었으므로, **텍스트 형태의 메시지도 사전에 이미지로 제작해 전송해야 함**
- `이미지 파일 전송` 및 `이미지 스케줄 관리` 기능이 구현되어 간단하게 **이미지를 업로드하여 문구와 이미지를 자유롭게 표출 가능**

### 5. 문의사항
- 다음의 내용이 정해졌는지 문의
	- 실제 전송할 문구 개수
	- 실제 전송할 문구의 내용(문자, 이미지, 효과 등)
	- 설치될 전광판 크기(가로/세로 픽셀수, 픽셀간 간격 등)
