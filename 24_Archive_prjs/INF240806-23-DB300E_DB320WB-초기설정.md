#### ESP32 Binary File Update
> [!note] 펌웨어
> Bootloader 펌웨어: DB300-BOOT V2.0.0.bin
> User 펌웨어: DB300E_V2.0.4.bin, DB320WB_V2.1.1.bin
- DB300E/DB320WB를 JP1 단락 후 TTL로 전원공급
- flash_download_tool_3.9.6 실행
	- ChipType: `ESP32`
	- WorkMode: `Develop`
	- LoadMode: `UART`
	- `OK`
		- ![[esp32_1.png]]
	- 공통
		- `DB300-BOOT V2.0.0.bin` @`0x00000`
		- SPI SPEED: `80MHz`
		- SPI MODE: `QIO`
		- DoNotChgBin `체크`
	- 유저 펌웨어
		- DB300E: `DB300E_V2.0.4.bin` @`0x10000`
		- DB320WB: `DB320WB_V2.1.1.bin` @`0x10000`
	- `START`
		- ![[esp32_2.png]]
			

#### DB300E/DB320WB Mac Address 입력
- 출력한 Mac Address 바코드를 DB300E/DB320WB 뒷면에 부착
- DB300E/DB320W를 TTL로 연결
- 다빛채 > `메뉴모음` > 메뉴모음 창 > `통신설정` > Serial 통신 > `포트열기` > `컨트롤러 연결`
	- F10 > `MacAddress 설정` > `DB300E/DB320WB 부착한 Mac Address` > `전송`
	- `읽기`를 눌러 전송 확인
	- `닫기`
		- ![[macaddress_1.png]]
