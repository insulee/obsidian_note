
#### 배경이미지 스케줄 전송 
- 보드기능설정 > 4. BH1(TTL/RS485) Function > 8Pin Input(BIT BGSch)
- 최대 64개의 이미지 파일 중 **10개까지 선택 가능**
- 이미지 전송 패킷: `![0020abc def ghi!]`
	- `abc def ghi`: 전광판에 표시할 배경 이미지 번호
- 응답 패킷
	- `![00200!]`: 성공
	- `![0020F!]`: 실패
- 예시
	- `![0020001 009 015 030!]`: 배경 이미지 목록 중에서 1번, 9번, 15번, 30번을 스케줄에 등록
- DBPS
	- 다중 배경이미지 선택

#### 표출속도설정
 - 속도설정 패킷: `![0054 ab!]`
	- `ab`: 전광판에 표시할 배경 이미지 속도
- 응답 패킷
	- `![00540!]`: 성공
	- `![0054F!]`: 실패
- 예시
	- `![0054 99!]`: 99(표출 속도가 가장 느림)
	- `![0054 01!]`: 1(표출 속도가 가장 빠름)
	- `![0054 00!]`: 표출 속도 설정 값을 적용하지 않고, 기존 스케줄의 표출 속도를 적용

#### 점멸횟수
- 점멸횟수 패킷: `![0055 ab!]`
	- `ab`: 색상깜빡이기 적용시 점멸 횟수 (1회~20회)
	- 기본값은 8회로, `00` 또는 `21` 이상의 숫자는 8회로 설정
- 응답 패킷
	- `![00550!]`: 성공
	- `![0055F!]`: 실패
- 예시
	- `![0055 20!]`: 점멸효과 20회
	- `![0055 05!]`: 점멸효과 5회
	- `![0055 00!]`: 점멸효과 8회
	- `![0055 22!]`: 점멸효과 8회
- DBPS
	- 7. 입장효과 / 8. 퇴장효과 > 색상깜빡이기 > 깜빡임 횟수 > **드롭다운 버튼(1~20)**

#### 폰트 두께 계수
 - 폰트두께 계수 패킷: `![0056 XX YY!]`
    - `XX`: 가로 두께 계수 (10 ~ 30)
    - `YY`: 세로 두께 계수 (10 ~ 30)
- 응답 패킷
	- `![00560!]`: 성공
	- `![0056F!]`: 실패
- 예시
    - `![0056 00 00!]`: 두께 계수 조정 비활성화
    - `![0056 12 20!]`: 가로 두께 1.2배, 세로 두께 2.0배 적용
- DBPS
	- 폰트 전송 창 > **계수 설정 드롭다운 버튼**
