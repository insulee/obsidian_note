## 다빛솔루션 컨트롤러와 유진LED 컨트롤러 비교 테스트 보고서

---

### 1. 테스트 개요

- **테스트 컨트롤러:**
    - **다빛솔루션:** DB500 v12 (firmware: V3.1.2, 이하 DB500)
    - **유진LED:** UJM-Controller Ver 1.2 (이하 UJM)
- **테스트 프로그램:**
    - **다빛솔루션:** 다빛프로토콜시뮬레이터 V2.3.0
    - **유진LED:** LED Display Test Tool Ver 4.2D
- **테스트 환경:**
    - Ethernet 연결을 사용하며, 동일한 LED 모듈 (P3 2단6열)을 적용하여 비교
    - 동일한 환경에서 텍스트 표출, 표출 기능 및 기타 설정을 테스트
- **평가 항목:**
    - **텍스트 표출:** 실시간 메시지와 페이지 메시지 표출
    - **표출 기능:** 애니메이션 효과, 속도, 유지 시간, 문자 표출
    - **기타 설정:** 전광판 크기 설정, 폰트 전송, 표출신호 및 기타 부가 기능

---

### 2. 테스트 결과


**A. 텍스트 표출 (실시간/페이지 메시지)**

- **실시간 메시지:**
    - 자주 변경되는 데이터를 즉시 표출하여 긴급 정보 및 실시간 데이터 업데이트에 사용
- **페이지 메시지:**
    - 변경이 적은 문구를 무한 반복하여 표출
- **DB500:**
    - 실시간 메시지 전송 시 해당 효과가 지속되며, 페이지 메시지보다 우선적으로 표출
- **UJM:**
    - 실시간 메시지 효과 유지 중 페이지 메시지 전송 시 페이지 메시지가 표출되는 문제가 발생

**B. 표출 기능**

- **문자 표출:**
    - **DB500:**
        - 40여 가지의 다양한 효과 지원 및 최대 220자까지 표출 가능
    - **UJM:**
        - 2단6열 모듈에서 영문/숫자는 최대 24자, 한글은 최대 12자까지만 표출되며, 초과하는 문자는 표시되지 않음
- **한글 출력 문제:**
    - UJM은 한글이 홀수 바이트로 시작할 경우 일부 문자가 올바르게 출력되지 않음
    - 예를 들어, "1가나다라마바" 전송 시 마지막 글자 "바"의 일부가 제대로 내려가지 않는 문제 확인

**C. 기타 설정 기능**

- **DB500:**
    - 전광판 크기 설정, 폰트 크기 조절, 표출신호 설정, 보드 기능 설정, 화면 밝기 조절, 컨트롤러와 PC 시간 동기화, 전광판 온/오프 등 다양한 부가 기능 제공
- **UJM:**
    - 페이지 메시지 삭제, 전광판 온/오프 기능만 제공

---

### 3. 문제점

- **실시간 메시지 표출 문제:**
    - UJM의 경우, 실시간 메시지 효과가 유지되는 동안 페이지 메시지가 동시에 전송되면 페이지 메시지가 표출
- **표출 기능 한계 및 불안정성:**
    - UJM는 2단6열 모듈에서 영문/숫자 24자, 한글 12자만 표출되어 표출이 제한됨
    - 한글이 홀수 바이트로 시작할 경우 2단 표출 시 일부 문자가 올바르게 출력되지 않는 문제가 있음
- **기타 설정 기능 제한:**
    - DB500은 다양한 부가 설정을 지원하는 반면, UJM은 페이지 메시지 삭제 및 전광판 온/오프 기능 정도의 기본 설정만 제공됨

---

### 4. 결론


- 테스트 결과, 다빛솔루션의 DB500 컨트롤러는 유진LED의 UJM 컨트롤러에 비해 전반적인 표출 품질, 안정성, 그리고 사용자 편의성 측면에서 월등한 성능을 보임

- DB500은 다양한 효과와 설정 옵션을 제공하여 사용자 맞춤형 전광판 연출이 가능하며, 통신 연결 및 실시간 메시지 처리에서도 우수한 성능을 나타냄

- 반면, UJM은 고정된 네트워크 설정, 제한된 문자 표출, 불안정한 텍스트 출력, 그리고 설정 기능의 제약 등으로 인해 기본적인 기능인 전광판 텍스트 표출에 문제가 발생
