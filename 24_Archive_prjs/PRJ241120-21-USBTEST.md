
#### 테스트환경
- 프로그램: 다빛채 V2.4.8
- 컨트롤러: DABIT600TV12(v2.6.8-1120)

#### 테스트 순서
1. 컨트롤러1에 .pla 또는 .bgp 업로드
2. dabitche > Memory > Write > 파일생성 확인
3. 생성파일 USB로 복사
4. .bin 파일로 확장자 변경
5. 컨트롤러2에 USB 삽입, 초기화 화면 나오면 USB 제거 후 적용사항 확인
