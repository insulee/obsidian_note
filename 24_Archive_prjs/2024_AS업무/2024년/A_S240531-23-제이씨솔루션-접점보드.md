## 원인 분석 및 해결 방안

#### 원인 분석
- 접점보드 연결시 BH1 function이 기본설정인 TTL/RS485로 되어 있음
- 이로 인해 데이터 케이블만 연결해도 불이 들어오는 것으로 예상
#### 해결 방안  
1. DABIT Protocol Simulator 실행
2. 설정 및 제어 > 고급설정 > 보드기능설정 > 2. Sub Command 
	1. `설정`
	2. 4\. BH1(TTL/RS485) Function > `8Pin Input(HEX)`
	3. `전송`
	4. `닫기`
		- ![[as_제이씨솔루션2.png|200]]
- 참고
	- 전송을 눌렀을 때, 접속이 안된다면 응답시간을 3초 이상으로 설정
	- 설정 및 제어 > 통신설정 > 응답시간설정 `3`sec > 닫기
		- ![[as_제이씨솔루션3.png|200]]
	- BH1(TTL/RS485) Function에서 8pin으로 세가지 옵션 선택가능
		1. `8Pin Input(HEX)`
		2. `8Pin Input (Number)`
		3. `8pin Input (BCD)`
		- 세가지 옵션에 관한 이론 및 return value - [[xINF240531-21-접점신호메세지표시]]
