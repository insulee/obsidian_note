### 원인 분석 및 해결 방안

#### 해결 방안  

> [!NOTE] 다빛넷에서 bmp파일을 만들어서 배경으로 전환

다빛넷 매뉴얼 기준
- bmp파일 제작 (다빛넷 매뉴얼 첨부2)
	- 전광판 크기 설정
		- 환경설정 > 화면설정
	- 그림판에서 파일 만들기
		- 표시파일작성 > 그래픽 이미지
		- 그림판 파일 작업
			- 연필 1px > 색상지정 > 그리기
			- 다른이름으로 저장 > BMP 그림 > Image 폴더에 저장
- bmp파일을 dat파일로 변환 (다빛넷 매뉴얼 4.6표시파일)
	- 표시파일작성 > 이미지 편집
	- 이미지 편집 > 이미지 고급편집> 배경이미지 불러오기 > 메인이미지 > ...
		- Select Image File > .bmp > 열기
	- 적용
		- Text Date File Save > .dat > 저장
- 표시목록 만들기 (다빛넷 매뉴얼 5.1표시목록 만들기, 5.2배경화면 표시목록 만들기)
	- 표시파일명 > 이미지 > dat 파일을 열고
	- 저장 > 파일을 .bgp로 선택하여 저장
	- 업로드 버튼을 클릭하여 전광판에 전송
- Dabit Protocol Simulator
	- HEX: 16. 배경이미지 선택 > 1번
	- ASCII: /B001
