### 요청사항
#### 1. 최신 펌웨어 적용 테스트 요청

현재 DB600과 DB500 컨트롤러는 **최신 펌웨어 V3.0.5**로 업데이트되었습니다. V3.0.5는 다음과 같은 주요 업데이트 내용을 포함하고 있습니다.

##### 1.1 주요 업데이트 내용 (첨부문서)
1. **스케줄 확인 패킷**(`PRJ241120-23-성화기술산업-스케줄확인패킷.pdf`)
	- 전광판에서 스케줄에 등록된 여러 파일(dat, daz, ani, anz) 을 순차적으로 표출
	- 각 파일의 **표출 효과 종료** 시, **시리얼 통신**을 통해 **종료 확인 패킷**을 전송 받음
2. **폰트 두께 설정 패킷**(`PRJ241125-21-성화기술산업-폰트두께조정.pdf`)
    - 폰트의 **가로 및 세로 두께**를 조정하는 기능을 구현해 기존 보간값보다 더 높은 사용성과 범용성을 제공
    - 두께는 사용자가 정의한 **두께 계수**에 따라 조정되며, 통신 패킷을 통해 설정 값을 전송하고 확인
3. **USB 전송 기능 추가**(`INF241129-23-USB설정.pdf`)
    - USB를 통해 다음 항목들을 손쉽게 전송할 수 있도록 개선
        - 펌웨어 파일
        - 컨트롤러 설정값(전광판크기, 표출신호, 폰트, 보드기능설정 등)
        - 배경 이미지 파일
        - 메인 스케줄 파일
##### 1.2 요청 사항
- 최신 펌웨어(V3.0.5)가 귀사 시스템에서 원활히 작동하는지 **테스트를 부탁**드립니다.
- 만약 문제점이 발견될 경우, 공유해주시면 해결방안을 찾아 최신 펌웨어를 사용할 수 있도록 지원하겠습니다.

#### 2. 발주 및 납기 일정 관리 협조
- 발주 요청 후 납기까지 최소 **1주일 이상 소요**되는 점을 고려하여, 적용 가능한 펌웨어 버전을 사전에 협의하여 원활한 공급 계획을 수립할 수 있도록 협조를 요청드립니다.
- 안정적인 납기를 위해 발주 일정과 관련된 사항은 가능한 빠르게 공유해 주시면 감사하겠습니다.

#### 3. AS 요청 내용 정리
- AS 요청 시 해당 컨트롤러의 문제되는 부분을 빠르게 찾고 해결하기 위해, **컨트롤러의 MAC 주소별로 AS 요청 내용을 정리**하여 아래의 양식과 같이 간단하게 보내 주시면 감사하겠습니다.
