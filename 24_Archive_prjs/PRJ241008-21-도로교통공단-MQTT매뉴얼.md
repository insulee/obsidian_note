#### Node-RED 설치
1. Node.js를 다운로드 받아서 설치
	- Node.js 웹페이지(https://nodejs.org/) 접속
	- Download Node.js(LTS) 다운로드
2. 정상 설치 확인
	- 명령프롬포트(윈도우키+R > cmd)
	- `node --version && npm --version`
	- 두 개의 버전이 표시되면 정상 설치
3. npm 명령어로 Node-RED 설치
	- 명령프롬포트(윈도우키+R > cmd)
	- `npm install -g --unsafe-perm node-red`
4. Node-RED 실행
	- 명령프롬포트(윈도우키+R > cmd)
	- `node-red`
