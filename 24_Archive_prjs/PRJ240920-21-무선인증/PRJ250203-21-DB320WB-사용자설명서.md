### 다빛솔루션 DB320WB 포트
![[db302wb_1.png|400]]
1. JTAG: 프로그램 플래싱 및 디버깅 기능
2. 접점신호(10pin) 포트
3. RS232 통신 포트
4. TTL 통신 포트 (컨트롤러 통해서 전원 공급)
5. Relay 포트
6. Sensor 포트
### 테스트 세팅(하드웨어)
- DB320WB (JTAG) <-- 2 pin
- DB320WB (접점신호) <--10pin--> 접점신호보드(DB200-IN8P24V-OUT5V v0.2)
- DB320WB (RS232) <--4pin--> PC/Laptop
	- SerialPortMon(통신 모니터 프로그램)을 통해서 데이터 전달 확인 
- DB320WB (TTL) <--4pin--> 전광판 컨트롤러(DB600T, 전원연결) <--플랫케이블--> LED 모듈(전원연결)
	- 전원 공급 및 데이터 전달
	- LED 모듈 표출
- DB320WB (Relay) <--4pin--> 2 Relay Module
- DB320WB (Sensor) <--5pin--> TEMP&HUM Sensor

### 테스트 세팅(소프트웨어)
#### Wifi
1. PC/Laptop > Wifi > AP-dabitnet 연결
2. DABIT Protocol Simulator
    1. 설정 및 제어 > `통신 설정` > 통신 설정 창
	1. DABITNet > `Search`
	2. `54-FF-82-80-00-01` 선택
	3. 아래의 설정으로 세팅
		- Local IP: `192.168.4.1`
		- Subnet Mask: `255.255.255.0`
		- Gateway: `192.168.4.1`
		- Port: `5100`
	4. `Add`
	5. `접속하기`
	6. `컨트롤러 연결`
	7. `닫기`
3. SerialPortMon (패킷전송용)
	1. `Connect`
	2. 연결형태: `TCP Client`
	3. DABIT Protocol Simulator에서 변경한 값
		- 원격 IP 주소: `192.168.4.1`
		- 포트번호: `5100` 
	4. `Connect`
	5. 단축키 메시지 편집 대화상자 (1초간격으로 24시간)
		- 문자열: `![000Hi Wifi!]`
		- 지연: `1000`
		- 반복: `86400`
		- `보내기`
4. SerialPortMon (디버깅용)
	1. `Connect`
	2. 연결형태: Serial(COM)
	3. 연결포트: COM1(실제 연결된 포트)
	4. `Connect`

#### BLE
1. PC/Laptop > Bluetooth 및 장치 > `dabit_ble` 선택 > 페어링됨 확인
2. 장치 관리자 > 포트(COM & LPT) > 2개의 Bluetooth 포트 확인
3. DABIT Protocol Simulator
    1. 설정 및 제어 > `통신 설정` > 통신 설정 창
        - `Serial` 선택
        - 통신 포트 설정: `Bluetooth 포트`
        - 통신 속도 설정: `115200`
        - `포트열기`
        - `포트닫기`
    2. `Bluetooth` > 블루투스 설정 창
        - 블루투스 이름: `dabit_ble`
        - 블루투스 비밀번호: `dabit_ble`
        - `BeginBLE`
        - 소프트웨어 하단의 로그 보기 > `![DIBD BLE OK!]` 확인
        - `닫기`
    3. `포트열기`
    4. `컨트롤러 연결` 
4. SerialPortMon (패킷전송용)
	1. `Connect`
	2. 연결형태: `Serial(COM)`
	3. 연결포트: COM3(BLE 연결 포트)
	4. `Connect`
	5.  단축키 메시지 편집 대화상자 (1초간격으로 24시간)
		- 문자열: `![000Hi BLE!]`
		- 지연: `1000`
		- 반복: `86400`
		- `보내기`
5. SerialPortMon (디버깅용)
	1. `Connect`
	2. 연결형태: `Serial(COM)`
	3. 연결포트: COM1(실제 연결된 포트)
	4. `Connect`
