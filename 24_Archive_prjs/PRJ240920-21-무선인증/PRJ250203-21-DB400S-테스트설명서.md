### 테스트 세팅(하드웨어)
- DB400 (P1) <-- Ethernet --> 데이터 전송 
- DB400 (P2) <-- usb c --> 전원공급
- DB400 (J1) <-- TTL --> 전원공급
- DB400 (BH1) <-- 16핀 케이블 --> LED 모듈 연결


### 테스트 세팅(소프트웨어)
1. DABIT Protocol Simulator
    1. 설정 및 제어 > `통신 설정` > 통신 설정 창
	1. DABITNet > `Search`
	2. `54-FF-82-02-02-27` 선택
	3. 아래의 설정으로 세팅
		- Local IP: `192.168.0.227`
		- Subnet Mask: `255.255.255.0`
		- Gateway: `192.168.0.1`
		- Port: `5000`
	4. `Add`
	5. `접속하기`
	6. `컨트롤러 연결`
	7. `닫기`
2. SerialPortMon (패킷전송용)
	1. `Connect`
	2. 연결형태: `TCP Client`
	3. DABIT Protocol Simulator에서 변경한 값
		- 원격 IP 주소: `192.168.0.227`
		- 포트번호: `5000` 
	4. `Connect`
	5. 단축키 메시지 편집 대화상자 (1초간격으로 24시간)
		- 문자열: `![000Hi DB400!]`
		- 지연: `1000`
		- 반복: `86400`
		- `보내기`
